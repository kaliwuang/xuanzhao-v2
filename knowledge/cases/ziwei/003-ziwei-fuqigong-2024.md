# 紫微案例 002: 男命夫妻宫凶星组合(桃花旺)

## 命主信息

- **性别**:男
- **案例类型**:紫微斗数姻缘分析
- **出处可信度**:★★★☆☆ (搜狐系列专栏)

## 关键星曜配置(摘要)

**夫妻宫特征**:
- 主星:**天相**
- 辅星:**铃星**(凶)
- → 经典"不爱老婆"组合

**关联星曜**:
- 桃花星旺
- 欲望强

## 作者判读(原文摘要)

> "不爱老婆,欲望强桃花旺,一定出轨?"

**实际判读**:**不一定出轨,但有出轨倾向/欲望**

## "为什么会这样"的因果链

```
紫微断姻缘:
1. 取用神(姻缘 → 夫妻宫)
2. 夫妻宫主星看本质 → 天相(辅佐、协调)
3. 辅星铃星 → 暗耗、隐藏的负面情绪
4. 桃花星旺 → 异性缘强
5. 综合 → 有出轨倾向但不一定真出轨
```

## 应验情况(原文未明示具体事件)

**这是模式判读而非事件判读**:
- 不预测"哪一年出轨"
- 而是判读"出轨倾向强度"
- 实战中:这种命主需要"主动管理欲望",否则容易出问题

## 判断准则提炼

- **ZW1 夫妻宫铃星 → 暗耗婚姻,有出轨倾向**(★★★★☆)
- **ZW2 桃花星旺 + 铃星 → 高风险组合**(★★★★☆)
- **ZW3 天相 + 铃星 = 表面和谐,内在暗流**(★★★☆☆)

## 关键 takeaway

**紫微"模式判读"vs"事件判读"**:
- 模式判读 = 看趋势(倾向、可能性)
- 事件判读 = 看具体(哪年发生)
- 实战中:模式判读 + 大运流年 = 事件应期

**这一类案例教我们**:紫微的优势在模式判读,不是事件判读。事件判读需结合八字/奇门等时运。
