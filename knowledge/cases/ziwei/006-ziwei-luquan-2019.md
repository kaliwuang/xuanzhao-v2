# 紫微案例 006: 禄权组合加持命盘(开挂命盘)

## 命主信息

- **性别**:男
- **出处可信度**:★★★☆☆ (网易订阅系列)

## 关键星曜配置

**禄权组合**:
- **禄**(化禄)= 主财、机会、机遇
- **权**(化权)= 主权、掌控、能力

**实战意义**:
- 禄 + 权 出现在关键宫位 → "开挂"命盘
- 典型组合:命宫、财帛宫、官禄宫有禄权

## 判断准则提炼

### ZW7 禄权组合 = 富贵命盘标志

**实战**:
- 命宫有禄或权 → 个性有魅力,有机会
- 财帛宫有禄或权 → 财源滚滚
- 官禄宫有禄或权 → 事业有成
- 多个宫位有禄权 → 大富贵命

### ZW8 禄权 vs 科忌

**实战对比**:
- 禄权多 = 富贵
- 科忌多 = 文雅但清贫
- 禄权 + 科忌 = 平衡型富贵(实战最好)
