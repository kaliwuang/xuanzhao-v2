## 判断准则提炼

### ZW11 好命盘的标志(至少 3 个)

实战中遇到下列任一 3 项以上:
1. **命宫有禄存或化禄** → 财源、机遇
2. **官禄宫有化权或化科** → 事业成就
3. **财帛宫有禄** → 财源滚滚
4. **夫妻宫有吉星** → 配偶助力
5. **父母宫有吉星** → 长辈荫庇
6. **田宅宫有吉星** → 家庭基础好

实战:好命盘不是"有财星就有钱",而是**多宫位吉星汇聚**

## 实战警告

**好命盘 ≠ 一帆风顺**:
- 大运走得不对 → 富贵可能迟到
- 后天努力不够 → 浪费好命
- 选择错误 → 错失机会

## 关键 takeaway

**好命盘的实战识别**:
- 单看一两个吉星不够
- 必须多宫位综合
- 实战中:看客户命盘,数"吉星覆盖的宫位数量"
- 3+ 个吉宫位 = 实战好命盘
- 1-2 个 = 一般
- 0 个 = 需要努力
