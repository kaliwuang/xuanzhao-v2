# 太乙案例 004: 历史国运判读(《南齐书》案例集)

## 来源

《南齐书·本纪第一·高帝纪上》中的史臣按语,引用《太一九宫占》推算历史事件。

## 历史案例(原文)

**1. 汉高五年(公元前 202 年)**
- 太一在四宫 → 主人与客俱得吉,先举事者胜
- → **刘邦破楚**

**2. 晋元兴二年(403 年)**
- 太一在七宫 → 太一为帝,天目为辅佐
- → **安帝被桓玄逼迫出宫**

**3. 元嘉七年(430 年)**
- 太一在八宫 → 关囚恶岁
- → **到彦之北伐初胜后败**

## 应期判断

古代太乙的应期 = **以年为单位**:
- 太一在 X 宫 → X 宫的"主事"应在某年发生
- 历史案例已"事实验证"

## 判断准则提炼

### TY10 太乙断国运(以年为单位)

实战:
- 太一在四宫(东南)→ 该年多吉事
- 太一在七宫(西北)→ 该年凶事,主政变
- 太一在八宫(东北)→ 该年关囚(事败)

### TY11 太乙主客之分

实战:
- 陈兵原野(战乱)→ 先动为客,后应为主
- 安居之势(和平)→ 先动为主,后应为客

## 与现代占卜的区别

古代太乙 = 国运判读
现代太乙 = 个人事件(约会/生意/求职)

## 关键 takeaway

**太乙的实战分层**:
- 古代:国运(以年为单位)
- 现代:个人事件(以日为单位)
- 实战中:太乙适合大型事件判断(生意/合作/求职)

**太乙 vs 八字/奇门**:
- 太乙:大事件(国家/企业/大型合作)
- 八字:个人长期(终身)
- 奇门:个人短期(单事件)
