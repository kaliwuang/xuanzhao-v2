# 太乙案例 002: 客户被欠薪 6 万测官司

## 求测信息

- **关系**:求测人(被欠薪员工)
- **事项**:被公司拖欠 6 万多工资,欲起诉
- **结果**:求测人胜诉

## 起局分析(原文)

- **主动为始击(求测人)**,被动为文昌(公司)
- **求测人临君基**,公司临臣基 → 君臣颠倒
- **太乙在艮为法官**,火生土 → 法官偏向求测人
- **四神为上下流通** → 象意为贿赂(实际未必发生)

## 应期与数字

- **应数**:二五八,旺取八 → 应数为 8000 左右
- **预测**:即使不送礼,法官也会偏向求测人

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 君臣颠倒 | 求测人胜诉 | ✅ |
| 法官偏向求测人 | 胜诉 | ✅ |
| 应数 ~8000 | 实际赔偿额近 8 千 | ⏳ 细节未明 |

## "为什么会变成这样"的因果链

```
太乙断官司:
1. 主动=求测人,被动=对方
2. 君臣位 = 求测人 vs 对方的位阶
3. 君臣颠倒 → 上下位错位 → 弱势方实际有利
4. 太乙在艮=法官 → 火生土 → 法官倾向
```
