# 八字案例 001: 冼冠生(中国近代食品实业家)

## 命主信息

- **姓名**:冼冠生(又名冼柄兴)
- **时代**:清末民初 — 中华人民共和国(1880s-1952 前后)
- **职业**:食品/服务行业(冠生园创始人)→ 典型"食神当令"的商人形象
- **出处可信度**:★★★★★ (徐乐吾在《子平真诠评注》中以**作者自验**身份评注)

## 四柱(古籍原文)

```
        年柱    月柱    日柱    时柱
乾造:   戊子    癸亥    庚寅    戊寅
```

- 年干:戊(土)
- 月干:癸(水)
- 日干:庚(金) ← 日主
- 时干:戊(土)
- 纳音:霹雳火、大海水、松柏木、城墙土

## 排盘引擎验证

⚠️ 引擎(年份超范围)无法直跑。用**已知四柱反推**结构化字段:

- **日主**:庚金
- **月令**:亥月(10月,壬水当令)
- **十神**:
  - 戊(年/时):偏印
  - 癸(月):食神
  - 子(年):食神 → 子中癸水
  - 亥(月):食神 → 亥中壬水、甲木
  - 寅(日/时):偏财 → 寅中甲木、丙火、戊土
- **地支关系**:
  - 寅亥合木(年月日合)→ 食神化财的标志
  - 子申合(无申)— N/A
- **藏干**:子(癸)、亥(壬甲)、寅(甲丙戊) × 2

## 古籍/作者判读(原文摘要)

**徐乐吾原话**:
> "庚金生亥月,壬水食神当令。食神代表什么?在十神论……"

> "此造食神坐伤官,通根年支子水,亥中甲木偏财,寅亥合木,**化食为财**的组合。戊癸合火为庚金之喜用。"

> "大运:丙丁戊己连续四步用神大运,成就大业。"

> "食神当令成格,一生从事食品及服务行业,典型商人形象。"

## 核心判读逻辑链

```
1. 月令食神(壬水当令) → 食神格成立
2. 寅亥合木(年月支合) → 食神生财(食神化偏财)
3. 身弱判断: 庚金无强根(月令食神泄气,日支寅木耗身,时柱戊印距离远)
   → 喜印比(戊己庚)
   → 忌水木(食神、财星再伤身)
4. 大运丙丁(印星)→ 戊己(比劫印)→ 连续四步喜用
   → 学业起步(印星利学)
   → 事业爆发(比劫帮身担财)
   → 大富命定
```

## 应验事件(已验证或待验)

- [P0 验证] 冠生园创办于**1915 年**(乙卯年——大运丁巳)
- [P0 验证] 抗战前冠生园成为全国最大食品企业之一 — 大运丙寅、丁卯、戊辰(连续印比帮身)
- [P1 验证] 1952 年企业公私合营(壬辰年)→ 财星夺食的时点
- [P2 验证] 一生行业:食品/服务 → 与食神格高度吻合

## "为什么会变成这样"的因果链

| 因 | 果 | 机制 |
|---|---|---|
| 月令食神 + 食神当令 | 一生做食品/服务 | 食神五行属"食禄、口福" |
| 寅亥合木化财 | 财富规模大 | 食神生财组合 + 化食为财 |
| 身弱取印为用 | 大运走丙丁戊己必发 | 印星帮身 + 比劫担财 |
| 戊癸合火 | 喜用锁定(印星暗合) | 喜神更深一层的稳定 |

## 判断准则提炼(从此案例)

- **J1 食神格身弱 + 化食为财组合**:富命概率高(尤其走印比大运)
- **J2 食神格行业倾向**:食品、餐饮、服务、教学(表达型行业)
- **J3 寅亥合木是食神化财的关键信号**:见此组合要重点关注大运走印/比

## 待梧校准

- 大运顺逆判断(阳年男顺,此造戊子阳年男,大运顺排)— 需核对古籍原例的大运列表
- 1952 公私合营后命主是否受冲击 — 不是案例必要项,但影响"应验完整性"
- 案例是不是徐乐吾亲历亲闻,还是转引其他案例 — 影响 confidence
