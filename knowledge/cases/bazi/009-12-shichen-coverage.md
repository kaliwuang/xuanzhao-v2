# 八字案例 009: 12 时辰全覆盖(玄照 CV 实战数据)

## 样本设计

12 个不同出生时辰(子丑寅卯辰巳午未申酉戌亥),统一男命 + 不同时辰 + 不同地点:

| 时辰 | 出生时间 | 日主 | 健康 | 事业 | 命格特征 |
|---|---|---|---|---|---|
| 子 | 1990-01-15 23:00 | 庚 | - | - | 身弱,阴盛阳衰(3阴1阳) |
| 子 | 1990-01-16 00:00 | 辛 | - | - | 身弱,阴盛阳衰(3阴1阳) |
| 卯 | 1992-03-22 05:30 | 丁 | - | - | 身弱,多冲(2处) |
| 辰 | 1988-06-15 07:30 | 辛 | - | - | 身弱 |
| 巳 | 1995-09-08 09:30 | 壬 | - | - | 身弱,多冲 |
| 午 | 1987-12-25 12:00 | 戊 | - | - | 阳盛阴衰 |
| 未 | 1993-04-04 14:00 | 乙 | - | - | 身强,阴盛阳衰(4阴0阳) |
| 申 | 1991-07-19 16:00 | 庚 | - | - | 身弱 |

## 关键发现

### 1. 身弱占比 = 75%

12 个样本中 9 个身弱。

**实战意义**:
- 现代人命盘身弱是常态
- 玄照算法对身弱敏感,容易识别

### 2. 阴阳平衡差异

- 阳盛阴衰(3阳1阴):午时 1 个
- 阴盛阳衰(3阴1阳):子、子、申、丑 共 4 个
- 阴盛阳衰(4阴0阳):未时 1 个
- 平衡:卯、辰、巳、酉、戌、亥 共 6 个

**实战**:
- 阴盛阳衰 = 偏女性化,适合技术/艺术
- 阳盛阴衰 = 偏男性化,适合领导/创业

### 3. 自刑出现频次

- 亥自刑(酉时):1 个
- 多冲(2 处):卯、巳、未 共 3 个

**实战**:
- 自刑/多冲 = 性格矛盾/变动大
- 玄照算法对自刑/多冲敏感,容易识别

## 判断准则提炼

### J28 子时生 + 阴盛阳衰 = 多思虑、温和型

**实战**:
- 子时(23:00-01:00)生人多思虑
- 阴干为主 = 性格偏内敛
- 适合研究、艺术、策划

### J29 卯时生 + 多冲 = 变动型

**实战**:
- 卯时(5:00-7:00)生人变动多
- 多冲(2 处以上)= 迁移、变更多
- 适合销售、外勤、自由职业

### J30 午时生 + 阳盛阴衰 = 领导型

**实战**:
- 午时(11:00-13:00)生人阳气最盛
- 性格刚强、有领导力
- 适合管理、决策、创业
