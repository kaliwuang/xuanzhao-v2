# 八字案例 002: 伍廷芳(清末外交官、法学家)

## 命主信息

- **姓名**:伍廷芳(Wu Tingfang,1842-1922)
- **职业**:清末外交官、北洋政府外交总长、司法总长,法学家
- **历史地位**:中国近代外交/法律现代化先驱,《万国公法》中文译者
- **出处可信度**:★★★★★ (徐乐吾亲自评注过)

## 四柱(古籍原文)

```
        年柱    月柱    日柱    时柱
乾造:   壬寅    丁未    己卯    乙亥
```

## 排盘引擎验证

- **日主**:己土
- **月令**:未月(6月,己土当令 — 但未中藏己土、丁火、乙木)
- **十神**:
  - 壬(年):偏财
  - 丁(月):正印
  - 乙(日/时):七杀 × 2
  - 寅(年):偏财 → 寅中甲木
  - 未(月):正印 + 己土
  - 卯(日):七杀
  - 亥(时):偏财 → 亥中壬水、甲木
- **地支关系**:
  - 亥卯未三合木局(年月日三合,完全成局)— 此为**从格判断的关键**
  - 寅亥合木(年时合)→ 加强木局

## 古籍/作者判读(原文摘要)

**徐乐吾评注**:
> "己生未月,干透丁火,支会亥卯未木局;丁合壬化木,丁未两字皆化为木,**从煞格**成。"

> "此造也是近代外交人物伍廷芳的八字。"

**争议**:
- 有研究者质疑从格门槛低,认为可能是普通正印格而非从格
- 主流派:《子平真诠》认为三合化局 + 印化从煞,可成从格
- 江湖派:认为从未月透丁不可从,只能算正印格

## 核心判读逻辑链

```
1. 月令未土本气为己 → 但地支三合木局(亥卯未),木局成化
2. 天干丁壬合 → 化木
3. 整个命局木势强盛,日主己土被克泄交加 → 顺势从木
4. 从煞格:性格服从权威、不固执,反而能成为"被器重者"
5. 外交事业:从煞格善于顺应大势(国际局势),故能成为外交家
```

## 应验事件

- 1896 年(丙申年)任驻美公使,开启外交生涯
- 辛亥革命后任南京临时政府外交总长、司法总长
- 一生服务于朝廷与革命党之间 → 从煞格的"顺势"特征
- 1922 年病逝(壬戌年,戌冲辰未?需查实)

## "为什么会变成这样"的因果链

| 因 | 果 | 机制 |
|---|---|---|
| 亥卯未三合木局 + 丁壬合 | 从煞格成立 | 日主被克泄至无力反抗 |
| 从煞格 | 善于顺势而为,不固执 | 不强行克煞反为吉 |
| 喜用:木(从煞之煞即喜神) | 一生涉外/与"他者"打交道 | 外交 = 与外部势力的对话 |
| 用神:印星丁火 | 法学家(法为"正印"之体现,印主文、法、权) | 法律即规则,印星护身 |

## 判断准则提炼

- **J4 亥卯未三合木局 + 月令土被化** → 从格可能性极高
- **J5 从煞格 + 印绶透干** → 从格中的"化格"变体,可富可贵
- **J6 从格行业倾向**:外交、法律、政治、服务(顺势而为的行业)

## 待梧校准

- 此造是否真从格?还是正印格?两派观点不同
- 若从格成立,则"甲乙木"是大用神——但他没从木
- 外在大运走向需核对古籍原例

## 反例风险

- 同样亥卯未三合 + 丁壬合,有从格破格反贫的案例(印比突然出现)
- 革命党与清廷双重身份,正是"从煞"的双面性 — 应验
