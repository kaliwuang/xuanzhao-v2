- 年柱庚午:金火,身旺
- 月柱己卯:土木

- 但个人能力/命运决定能不能接住

```
1. 庚金日主 + 庚金年柱 + 庚辰日柱 → 身旺
2. 卯月木旺 + 甲木时柱 → 财星强
3. 身旺 + 财星强 → 富命格局
4. 但午火(官杀) + 卯木(财) → 财官相生
5. 父辈有产业 → 财星为用,得到传承
6. 但"辛苦" → 命局带刑冲或大运走克
```
