# 八字案例 011: 乙木日主 · 藤萝系甲 · 几千万级别(微信公众号文章)

## ⚠️ 这案例入库的真实价值

**这是反面案例,不是正向案例**。

微信公众号文章《乙木日主,藤萝系甲,乙庚合化,富贵之命》给出:
- 一个乙木男命的逐大运流年分析
- 最终结论"**整体财富还是有几千万级别**"
- 一系列针对 2026-2029 流年的具体预测

**问题**:这是**自媒体单方面论述**,命主没有反馈"对/错",**完全不可验证**。

玄照的算法偏置库里早就标了:**财运判断 70% 偏吉**(1226+ 样本)。这种"几千万级别"的自媒体判断大概率属于偏置,不是真预测。

**入库目的**:
1. 记录"自媒体公开预测的盲点" — 看似有理有据,实际是不可证伪的判断
2. 提醒玄照:对客户的"我能赚多少"问题要标"不可预测",不要像这类文章给具体数字
3. 作为反向案例触发器:任何关于"财富几千万"的预测都要降信心

## ⚠️ 反向价值 — 这案例不能直接用

### 问题 1:数据不完整

文章没给完整八字,所有判断都建立在不完整数据上。**这是命理自媒体常态**。

### 问题 2:几千万级别是判断不是事实

- 月柱有"正财"是事实
- "月柱正财千万级别"是从事实跳跃到数字,无依据
- "整体几千万"是综合断言,无量化推导

### 问题 3:自媒体命理的常见手法

观察文章套路:
1. 用专业术语(食伤生财、藤萝系甲) → 显得专业
2. 流年逐个预测 → 显得精确
3. 末尾劝世 → 显得中肯
4. **核心数字(几千万)放在中间 → 看似有数据支撑**

玄照如果照搬这套,会变成跟外面算命 App 没区别的工具。

### 问题 4:无法证伪

命主没说话,所有人只能信作者推断。**真伪永远无解**。

## 玄照应该做的(对此类案例)

### 1. 接受的部分

- "**乙庚合**" 是真经典格局 → 留作格局知识
- "**食伤生财**" 是真流通路径 → 留作判断规则
- "**2026 丙午卯午相破**" 是真神煞信号 → 留作流年规则
- "**伤官生财**" 大运方向 → 留作事业判断规则

### 2. 拒绝的部分

- **"几千万"** 具体数字 → 不输出
- **"官方背景"** 具体单位 → 不输出
- **"公检法/消防"** 具体推荐 → 不输出

### 3. 输出格式

玄照给客户的应该是:
```
事业方向:身弱借势,食伤生财流通好,适合借平台发展
流年(2026):有冲,事业不稳,留意身体
不输出:具体单位 / 具体收入数字 / 具体人生路径
```
