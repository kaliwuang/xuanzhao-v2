# 六爻案例 001: 某男考博通知书应期推断

## 命主/求测者信息

- **关系**:朋友家孩子(非命主本人,家人代测)
- **事项**:考国外博士,问能否通过、何时收到通知书
- **求测时间**:未明确(从推断应为考后等通知期间)
- **结果**:通过,**13 日壬午日**收到通知书

## 起卦信息(原文未给完整起卦时间,需桐补)

⚠️ 此案例为六爻"事件卦",**没有八字排盘**,只有起卦时间和卦象。

## 作者判读(原文摘要)

**推断**:
1. **能通过** — 推断逻辑应基于官鬼持世/应爻生世/世应关系
2. **13 或 14 日**收到通知书 — 推断逻辑应基于:
   - 13 日为壬午日(壬为官鬼、午为文书?)
   - 14 日为癸未日(未为木库生火?)

## "为什么会变成这样"的因果链

```
六爻应期断的核心逻辑:
1. 求测事件 → 取用神(考博 → 父母爻代表文书/通知书)
2. 应期 = 用神旺相之月日
3. 父母爻持世/动 → 文书主动来
4. 应期通常为:用神临值/临生/临旺之日
5. 壬午日(13日) → 午火生父母爻 → 触发应期
```

## 判断准则提炼

- **L1 考博/考试应期** → 看父母爻(文书)
- **L2 应期日判断** → 父母爻临值/生/合之日
- **L3 推断应期一般给 2-3 个候选日** → 选最旺者
