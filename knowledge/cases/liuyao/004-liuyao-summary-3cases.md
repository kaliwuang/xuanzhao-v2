## 综合判断准则(L1-L8)

### L1 考博/考试应期 → 看父母爻(文书)
### L2 应期日判断 → 父母爻临值/生/合之日
### L3 应期推断一般给 2-3 候选日
### L4 官鬼弱 → 入职/求职不利
### L5 世应相克 → 求测者与目标冲突
### L6 五爻乱动 → 事情多变数

### 新增 L7-L8
### L7 六爻断求职 → 父母爻(机会) + 官鬼爻(职位)
### L8 六爻看桃花 → 子孙爻

## 六爻实战比较

| 维度 | 六爻 | 八字 | 奇门 |
|---|---|---|---|
| 时间单位 | 事件(时/日) | 终身(年) | 事件(时) |
| 用神 | 六亲(父母/官鬼/...) | 十神 | 时干/格局 |
| 应期精度 | 高 | 中 | 高 |
| 实战场景 | 单事件 | 终身 | 单事件 |

## 实战建议

```
1. 客户问单事件(面试/考试/出行) → 六爻 或 奇门
2. 客户问长期趋势 → 八字
3. 应期精准 → 六爻(父母爻临值)
4. 整体状态 → 八字
```
