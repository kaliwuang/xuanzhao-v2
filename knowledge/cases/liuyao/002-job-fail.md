# 六爻案例 002: 男问能否成功入职某公司

## 求测者信息

- **关系**:问测人(男)
- **事项**:三轮面试后等入职通知,起卦问能否入职
- **结果**:**未能入职**(对方公司拒绝)

## 关键价值

**这是"反向应验"案例** — 排除"幸存者偏差"。大量"应验"的案例可能只是"未应验的没人发",这个案例记录了断"不能"且实际"确实不能"的情况,价值极高。

## 作者判读(原文摘要)

**断卦结果:不能入职**,依据:
1. 官鬼弱
2. 世应相克
3. 五爻乱动

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 不能入职 | 公司拒绝 | ✅ |

## "为什么会变成这样"的因果链

```
六爻断"不能"的逻辑:
1. 求测事件 → 取用神(入职 → 官鬼代表工作/职位)
2. 官鬼弱 → 职位不旺,无成
3. 世应相克 → 求测者与所求之事对立(性格/命格/时机不匹配)
4. 五爻乱动 → 多方变数,事情不稳
5. 综合 → 不能入职
```

## 判断准则提炼

- **L4 官鬼弱 → 入职/求职不利**
- **L5 世应相克 → 求测者与目标冲突**
- **L6 五爻乱动 → 事情多变数,不宜强求**

## 反例风险

- 单案例无法证明"官鬼弱 = 不能入职"普适
- 官鬼弱但其他条件齐备时,仍有可能成
