# 六爻案例 003: 1985 年男测回老家找工作(六爻事件卦)

## 求测信息

- **关系**:求测人(1985 年生男)
- **事项**:回老家找工作
- **结果**:待梧补全应验

## 起卦信息

- 起卦时间未明
- 用神:父母(文书)代表工作机会、官鬼代表职位

## 作者判读(原文摘要)

- 分析维度:工作选择、法律纠纷隐患、桃花运、身体健康
- 六爻卦象包含:世爻、应爻、子孙爻、官鬼爻

## "为什么会变成这样"的因果链

```
六爻断求职:
1. 取用神(求职 → 父母爻代表机会,官鬼代表职位)
2. 看世爻(求测人)与应爻(对方/工作地)关系
3. 看世爻旺衰 → 求测人状态
4. 看官鬼与世爻关系 → 能否被录用
5. 看父母爻 → 有无文书/合同
6. 综合 → 应期 + 吉凶
```

## 判断准则提炼

### L7 六爻断求职 → 父母爻 + 官鬼爻

实战:
- 父母爻旺 + 生世 → 有机会
- 官鬼旺 + 持世 → 能被录用
- 父母爻空亡 → 文书未到

### L8 六爻看桃花 → 子孙爻

实战:
- 子孙爻持世 → 桃花旺
- 子孙爻克官鬼 → 工作受影响
