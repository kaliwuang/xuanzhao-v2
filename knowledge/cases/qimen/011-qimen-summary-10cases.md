## 奇门实战比较

| 维度 | 奇门 | 八字 |
|---|---|---|
| 时间单位 | 单事件(时) | 终身(年) |
| 用神 | 时干+日干+格局 | 日主+十神+格局 |
| 应期精度 | 高(小时/天) | 中(年) |
| 实战场景 | 求职/考试/出行 | 婚配/子女/终身 |
| 短期事件 | **优势** | 弱 |
| 长期趋势 | 弱 | **优势** |

## 实战建议

```
1. 客户问短期事件(面试/考试/出行) → 奇门
2. 客户问长期趋势(婚配/事业) → 八字
3. 客户问重大决策 → 奇门(具体应期)
4. 客户问"何时发生" → 八字(年/月) + 奇门(日/时)
```
