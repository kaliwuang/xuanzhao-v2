# 奇门案例 007: 1990 年男命测投标

## 求测信息

- **关系**:求测人(1990 年 3 月生男)
- **事项**:投标能否成功
- **问测时间**:**2020-09-17 未时**
- **结果**:实际未中(从分析推断)

## 起局分析(原文)

1. **五不遇时** → 事情不顺利
2. **时干己临天芮病星 + 壬+辛腾蛇相缠** → 麻烦事缠身,易被人欺瞒
3. **日干落坤宫入墓** → 状态低,衰弱之象
4. **庚下临癸寅申相冲 + 杜门** → 事情有阻碍,难成
5. **景门(合同)逢空亡 + 白虎** → 投标成功希望不大
6. **值使门(投标机构)落土宫克坎水(年命庚)** → 投标机构克你,事不成

## 关键格局

**五不遇时**:
- 五不遇时 = 阴阳差错,百事不顺
- 实战中:起局遇五不遇时,事项"先天不利"

**日干入墓**:
- 求测人状态低
- 难以发挥

**多处空亡/入墓**:
- 景门空 + 白虎 = 合同出问题
- 值使门克年命 = 投标方不利
