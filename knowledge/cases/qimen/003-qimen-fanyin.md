# 奇门案例 003: 男 39 岁乙丑属牛测生意

## 求测信息

- **关系**:求测人(男,39 岁,乙丑属牛)
- **事项**:测生意经营
- **结果**:经营面临倒闭破财(具体年份未明,但"转变得太久")

## 起局信息

- **符下**:生门
- **落宫**:**震三宫飘浮、击刑**
- **格局**:**反吟局**

## 作者判读(原文摘要)

> "符下生门临震三宫飘浮、击刑,反吟局主经营面临倒闭破财,好在震三宫空亡勉强维持,转变得太久。"

## 关键格局解读

**反吟局**:
- 反吟 = 反复、变动、动荡
- 临生门 → 财运反复
- 临击刑 → 受伤、损失

**震三宫空亡**:
- 空亡 = 不实、勉强
- "勉强维持" = 还没完全倒闭
- 但"转变得太久" = 看不到好转迹象

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 经营面临倒闭 | 倒闭趋势 | ✅ |
| 破财 | 持续亏损 | ✅ |
| 空亡勉强维持 | 未完全倒闭 | ✅ |

## "为什么会变成这样"的因果链

```
奇门断生意:
1. 取用神(生意 → 生门代表财运)
2. 生门落震三宫飘浮 → 财运虚浮不实
3. 击刑 → 有损失
4. 反吟局 → 反复动荡
5. 空亡 → 暂时勉强
6. 综合 → 经营困难,长期亏损
```

## 判断准则提炼

- **QM7 反吟局 → 经营动荡 / 反复失败**(★★★★☆)
