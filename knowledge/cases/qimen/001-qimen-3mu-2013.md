# 奇门案例 001: 毛先生测儿子工作(1989 年生)

## 求测信息

- **关系**:毛先生(父代子测)
- **儿子**:1989 年生
- **事项**:找工作(目标:公检法单位)
- **起局时间**:**壬寅日 乙巳时**
- **结果**:**2013 年未找到公检法工作**(理论应期已过)

## 起局信息

- 日干:壬
- 时干:乙
- **三墓同断**:
  1. **开门落艮宫入墓**
  2. **时干乙奇落乾宫入墓临死门**
  3. **杜门值使落坤宫入墓临地盘庚**

## 作者判读(原文摘要)

> "壬寅日乙巳时起局,开门落艮宫入墓、时干乙奇落乾宫入墓临死门、杜门值使落坤宫入墓临地盘庚——三墓同断,**2013 年找不到公检法工作**。"

## 关键格局

**三墓同断**:
- 墓 = 入墓、停滞、办事不成
- 三处同时入墓 = 三重阻碍叠加
- → 应验难度极高

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 2013 年找不到公检法工作 | 命主反馈:确实未找到 | ✅ |

## "为什么会变成这样"的因果链

```
奇门断找工作的核心:
1. 取用神(找工作 → 开门代表工作/事业)
2. 开门落宫看:
   - 入墓 → 工作难求
   - 临庚 → 有阻碍(庚为阻隔)
   - 临值符 → 与领导相关
3. 三处同时入墓 = 三重不利叠加 → 应期难达
```

## 判断准则提炼

- **QM1 三墓同断 → 应事极难成**(置信度 ★★★★★)
- **QM2 开门入墓 → 求职不利**(经典奇门法则)
- **QM3 时干入墓 → 事情停滞**(经典奇门法则)
