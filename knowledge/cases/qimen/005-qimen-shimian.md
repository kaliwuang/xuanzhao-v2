# 奇门案例 005: 失眠风水案例(邹学亮)

## 求测信息

- **关系**:求测人(失眠问题)
- **事项**:失眠,寻求原因
- **结果**:准确推断出失眠的具体原因和应期

## 作者判读(原文摘要)

> "通过奇门盘推断:
> - 卧室窗帘有蛇形图案(应验)
> - 父亲患脑梗多年(应验)
> - 阳台堆放杂物造成风水问题"

**应期推断**:
- 失眠从 12 月开始
- 1-2 月加重
- 3 月特别明显
- (与未祭拜母亲、清明节未回去祭拜对应)

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 窗帘蛇形图案 | 求测者家确实有 | ✅ |
| 父亲脑梗多年 | 实际有 | ✅ |
| 阳台杂物 | 实际堆放 | ✅ |
| 12 月开始失眠 | 实际从 12 月开始 | ✅ |
| 1-2 月加重 | 实际加重 | ✅ |
| 3 月特别明显 | 实际 3 月最严重 | ✅ |

## "为什么会变成这样"的因果链

```
奇门断健康问题:
1. 取用神(失眠 → 病符星/死门)
2. 看对应宫位
3. 应期:对冲宫 + 引起问题的具体月份
4. 因果:具体物品(窗帘、阳台) → 风水问题
```

## 关键 takeaway

**奇门的"因果推断"能力**:
- 不仅能断"会失眠"
- 还能断"为什么失眠"(窗帘图案)
- 还能断"应期"(12月开始)
- 这是奇门 vs 八字的核心优势

**实战中**:
- 客户问"我身体怎么样" → 八字 + 紫微
- 客户问"我为什么失眠" → 奇门
- 客户问"我的病什么时候好" → 奇门(应期精准)
