# 奇门案例 004: 教师叶某测妻子生产

## 求测信息

- **关系**:教师叶某(求测人)
- **事项**:测妻子生产
- **起局时间**:**乙丑日 辰时**
- **结果**:第二天上午生产,生女孩,完全应验

## 起局信息

- 乙丑日辰时起局
- **用神**:以坤宫为产室,天芮为产母
- **应期推断**:
  - 性别:阴阳论男女 → 阴爻/奇数为女
  - 应期:对冲宫天盘干 → 第二天辰时(乙丑日 → 第二天是丙寅日,但实际是"第二天上午7-9点辰时")

## 作者判读(原文摘要)

> "准确推断:生产时间第二天上午 7-9 点辰时,性别女孩,第二天中午婆母前来报喜,完全应验。"

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 应期:第二天辰时 | 第二天上午 7-9 点 | ✅ |
| 性别:女孩 | 婆母报喜生女孩 | ✅ |

## "为什么会变成这样"的因果链

```
奇门断生产的核心:
1. 取用神(生产 → 坤宫为产室,天芮为产母)
2. 应期:看对冲宫天盘干 → 落到何时
3. 性别:阴阳爻位/天盘奇数
4. 时辰:起局时辰 + 1-2 天
```
