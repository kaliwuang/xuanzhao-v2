## 求测信息

- **关系**:求测人(男)
- **事项**:测姻缘什么时候会有
- **结果**:待梧补全应验

## 起局分析(原文摘要)

**关键格局**:
- **庚乙同宫落巽** + 临死门/白虎/天芮 → **迟婚晚婚象**
- **六合落震宫** + 丁+戊(青龙转光) → 现在不好,但**将来有转机**

## 关键格局解读

**庚乙同宫 = 男方女方同宫**:
- 一开始缘分有(同宫表示有连接)
- 但临死门/白虎/天芮 → 缘分被"凶神"压制

**六合落震 + 青龙转光**:
- 六合 = 婚姻宫
- 临青龙转光 → 现在暗淡,但有转机
- 震宫 = 东方,可能暗示发展方向

## 应验验证(原文未明确)

**作者声称**:此为典型"迟婚但有转机"案例

## "为什么会变成这样"的因果链

```
奇门断姻缘应期:
1. 取用神(姻缘 → 庚为男,乙为女,六合为婚姻宫)
2. 庚乙同宫 → 有缘
3. 但临死门/白虎 → 缘分被压
4. 六合落震 + 青龙转光 → 未来可期
5. 综合 → 现在没姻缘,过几年有
```

## 判断准则提炼

- **QM25 庚乙同宫 = 有缘**(★★★★☆)
- **QM26 同宫临死门/白虎 = 缘分被压**(★★★★☆)
- **QM27 六合 + 青龙转光 = 现在不好但将来有**(★★★★☆)

## 实战判断"姻缘应期"流程

```
1. 看日干(求测人)旺衰
2. 看乙奇(女方)落宫
3. 看六合(婚姻宫)落宫
4. 看是否有"凶格"(乙+辛、戊+庚 等)
5. 综合判读:有缘/无缘/迟婚/早婚
6. 应期:六合所临天盘干之时
```
