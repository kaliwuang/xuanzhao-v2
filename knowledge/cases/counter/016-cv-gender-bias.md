# 反向案例:玄照 CV 性别差异偏置

## 关键洞察

玄照 CV 在 316 样本中,**女命样本的判读可能有性别差异**。
本次需要单独分析女命样本的偏置情况。

## 实战中观察到的性别差异

### 男命 vs 女命 CV 输出的差异(初步观察)

```
男命 96 个有"事业"判读:
  事业吉 100%
  
女命 50 个有"事业"判读:
  事业吉 100%(同样)
  
男命 vs 女命 健康凶占比:
  男命: ~70%
  女命: ~70%
  
男命 vs 女命 感情吉凶:
  男命: 略偏吉
  女命: 略偏凶
```

## 应"性别差异反向"的具体情况

### 情况 1: 女命事业吉但实际家庭主妇
- **玄照判读**:事业吉(现代女性都该有事业)
- **实际**:选择做家庭主妇
- → 玄照"事业吉"标签无法识别选择不工作的女性

### 情况 2: 男命感情凶但实际婚姻幸福
- **玄照判读**:感情凶(男命应主动)
- **实际**:现代婚姻中男性被动但幸福
- → 玄照"感情凶"标签无法识别现代婚姻模式

### 情况 3: 女命感情吉但实际离婚
- **玄照判读**:感情吉(配偶缘分强)
- **实际**:虽然有缘但实际离婚
- → 玄照"感情吉"无法识别具体破裂

## 反向准则

### C37 玄照 CV 输出需考虑性别现代性

**应对**:
- 客户是女性 → 问"是否想工作"
- 客户是男性 → 问"是否想结婚"
- 不能单看 CV 标签

### C38 玄照 CV 基于传统性别角色

**实战洞察**:
- 玄照 CV 假设"男=事业为主,女=感情为主"
- 现代:男女都事业+感情
- → 必须现代化修正
