# 反向八字案例 006: 食神格身强走食伤大运(理论反例)

## 反向推断

**J1 修正版**:食神格 + 化食为财 + 走印比大运 → 富命(置信度 50-60%)

**反向条件**:**身强食神格 + 走食伤大运**

## 为什么身强食神格走食伤会贫困?

```
理论:
1. 身强不需要印
2. 食伤泄秀太过 → 才华外泄但没聚财
3. 大运再走食伤 → 持续泄秀
4. 身旺 + 食伤旺 = 才华横溢但贫困

实战例:
- 知名艺术家、作家、诗人(死后出名)
- 穷困潦倒的画家、音乐家
- 有才但赚不到钱
```

## 应"贫困但有才"的具体组合

```
1. 身强食神格 + 走食伤运(典型穷艺术家)
2. 身强食神格 + 财星弱(才华不换钱)
3. 身强食神格 + 印星坏(没人赏识)
4. 身强食神格 + 比劫夺财(合作分利)
5. 身强食神格 + 食神入墓(才华被压)
```

## 实战中的判断方法

**客户问"我会不会富"时,如果有食神格**:
- **身强** → 问"你是不是喜欢艺术/创作"
- **走食伤大运** → "短期难聚财,但有才"
- **走印比大运** → "可能大器晚成"

**实战建议**:
- 食神格 ≠ 一定富
- 身强食神格 + 食伤运 = 才华型贫困
- 身弱食神格 + 印比运 = 实业型富命

## 判断准则修订

- **J1 食神格富命**:置信度 50%(原 70%)
- **C16 身强食神格 + 食伤运 = 才华型贫困**(新增)

## 关键 takeaway

**"X = Y"的判断永远要看身强弱 + 大运**:
- 即使"食神格"是吉格
- 身强 + 食伤运 = 穷
- 实战中:**讲条件,不下定论**
