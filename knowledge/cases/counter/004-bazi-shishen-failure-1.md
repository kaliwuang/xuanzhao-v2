# 反向八字案例 004: 食神格但实际贫困(理论反例)

## 案例性质

**这是基于 J1 准则推断的反向案例**,不是真实命主。

**推断逻辑**:
- J1:食神格身弱 + 化食为财 + 走印比大运 → 富命
- 反向:同样食神格,但**缺少某个条件** → 不富

## 推断的反例条件

### 反例 1: 食神格身强 + 印比不帮身 → 食伤泄秀太过

**理论**:
- 身强不用印,反而食伤泄秀太过
- 才艺外露,但不聚财
- 适合艺术家,但难大富

**实战**:身强食神格 → "有才但穷"型常见

### 反例 2: 食神被枭神夺食 → 才华被压制

**理论**:
- 偏印(枭神)夺食
- 有想法但总被打压
- 适合独处创作,难富贵

**实战**:枭神夺食 → 创业多挫败

### 反例 3: 食神格 + 大运走财官 → 食神制杀失败

**理论**:
- 食神制杀需要大运走印比/食神/伤官
- 若大运走财官 → 食神被财星坏印/被官杀克制
- → 应运不吉,反而破败

**实战**:大运走错运,食神格从吉变凶

## 应"未富"的具体条件清单

```
食神格不富的常见组合:
1. 身强 + 印比不帮身
2. 枭神夺食
3. 食神被冲(寅申冲、申酉冲等)
4. 大运走财官运
5. 时柱空亡(老无所依)
6. 食神入墓(食神被收藏,不发)
```
