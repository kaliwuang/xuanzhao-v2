# 反向八字案例 013: 事业"吉"但实际贫困(理论反例集)

## 关键洞察

**玄照 CV 在 316 样本中事业"吉" = 100%** —— 严重算法偏置,事业吉标签无意义。

实战中:**所有被告知"事业吉"的人,实际可能贫困**。

## 应"事业吉但实际贫困"的 6 种典型情况

### 情况 1: 身弱财官旺
- **命盘**:日主身弱 + 财星/官星太旺
- **玄照判读**:事业吉(有官/有财)
- **实际**:**扛不住财官** → 想做事但做不成

### 情况 2: 食神格身强(才华型)
- **命盘**:身强 + 食神格 + 走食伤大运
- **玄照判读**:事业吉(才华出众)
- **实际**:**才华横溢但贫困**(典型穷艺术家)

### 情况 3: 命中无财星
- **命盘**:有官星/印星但**无财星**
- **玄照判读**:事业吉(有方向)
- **实际**:**有名无利**(典型:教授/研究员)

### 情况 4: 比劫夺财
- **命盘**:身强 + 比劫多 + 财星被劫
- **玄照判读**:事业吉(有合作)
- **实际**:**合作分利,到手少**(典型:小老板)

### 情况 5: 印星坏财
- **命盘**:印星旺 + 财星弱
- **玄照判读**:事业吉(有学历)
- **实际**:**学历高但穷酸**(典型:清贫学者)

### 情况 6: 命中凶格太多
- **命盘**:多个凶格叠加
- **玄照判读**:事业吉(有转机)
- **实际**:**多次失败,负债**

## 反向准则

### C34 玄照"事业吉"标签基本无意义

**实战**:
- 永远不要直接说"事业吉"
- 应改为:"事业有方向/有机会"
- **必须看细分维度**(收入/创业/升迁)

### C35 事业吉 ≠ 收入吉

**实战**:
- 事业"吉"但收入可能"凶"(无名无利)
- 事业"吉"但创业可能"凶"(才华不等于商业)
- → 必须分开看

### C36 玄照的事业判读 = 方向判断,不是结果判断

**实战**:
- 事业吉 → "有方向"
- 事业凶 → "方向迷茫"(玄照几乎不出)
- 收入高低 → 看财运
- → 两个维度分开

## 关键 takeaway

**"事业吉"标签 = 玄照的最大盲点**:
- 100% 吉 → 客户无法区分
- 必须给细分维度:**行业方向 + 收入水平 + 升迁概率 + 创业适配**

**实战建议**:
- 客户问"事业怎么样" → 答"有空间"或"有方向"
- 客户问"能挣多少钱" → 看财运(70% 吉,但实际收入有差异)
- 客户问"能升职吗" → 看大运 + 官星
