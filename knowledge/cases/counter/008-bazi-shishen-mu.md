# 反向八字案例 007: 食神格入墓(理论反例)

## 反向推断

**食神格理论**:食神主才华、福气、表达。

**反向条件**:**食神入墓(食神在墓库中)**

## 食神入墓的实战反例

```
食神入墓 = 食神被压、被藏、不发挥
实战反例:
1. 有想法但总被环境压
2. 才华在公司里发挥不出来
3. 想辞职创业但没勇气
4. 表达欲被压制
5. 老好人但不出头
```

## 应"食神入墓不发挥"的具体组合

```
1. 食神在墓库中(辰戌丑未)
2. 食神被冲(寅申冲、申酉冲)
3. 食神被合(甲己合、乙庚合等)
4. 食神入空亡
5. 食神见枭(枭神夺食)
```

## 实战中的判断

**客户问"我是不是有才但发挥不出来"**:
- 食神格 + 食神入墓 = "是"
- 食神格 + 食神被冲 = "是"
- 食神格 + 食神被合 = "是"
- **典型表现**:在公司默默无闻,私下很多想法

## 化解方法

**实战建议**:
1. **找印星为用的环境**(印星是食神的源头)
2. **大运走食神/伤官运时再发挥**
3. **改名**(食神对应的字)
4. **调整行业**(适合食神的行业)

## 判断准则修订

- **C18 食神入墓 = 才华被压**(新增)
- **C19 食神被冲 = 才华不稳**(新增)
- **C20 食神被合 = 才华被牵制**(新增)

## 关键 takeaway

**食神格 ≠ 一定有才发挥**:
- 必须看食神是否被压(墓/冲/合/空)
- 实战中:**食神格是必要条件,不是充分条件**
