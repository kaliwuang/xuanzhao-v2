# 反向八字案例 015: 财运"吉"但实际低收入

## 关键洞察

**玄照 CV 在 626 样本中财运"吉" = 70%** — 大部分命主被判财运吉。
但实际:**约 30% 的"财运吉"命主,实际收入低**。

## 应"财运吉但实际低收入"的具体情况

### 情况 1: 身弱财旺(扛不住)
- **命盘**:日主身弱 + 财星太旺
- **玄照判读**:财运吉(财星强)
- **实际**:**扛不住财** → 财来财去,留不住

### 情况 2: 比劫夺财(分利)
- **命盘**:比劫多 + 财星被劫
- **玄照判读**:财运吉(有财)
- **实际**:**合作分利** → 实际到手少

### 情况 3: 财星入墓
- **命盘**:财星入墓库
- **玄照判读**:财运吉(有财)
- **实际**:**财被压** → 财发挥不出来

### 情况 4: 印星坏财
- **命盘**:印星旺 + 财星弱
- **玄照判读**:财运吉(有印)
- **实际**:**清贫学者** → 学历高但穷酸

### 情况 5: 食神泄秀(才华型)
- **命盘**:食神格身强 + 走食伤运
- **玄照判读**:财运吉(有食神生财)
- **实际**:**才华型贫困** → 才华有但钱少

### 情况 6: 财星空亡
- **命盘**:财星空亡
- **玄照判读**:财运吉(有时机)
- **实际**:**财落空** → 时机错过

## 反向准则

### C51 玄照"财运吉"≠ 实际收入高

**实战**:
- 永远不要告诉客户"你会发财"
- 应改为:"财运有 X 信号,需做 Y 努力"

### C52 财运判读需看身强身弱

**实战**:
- 身弱财旺 → "有财但需借势"
- 身强财弱 → "有空间但需努力"
- 身强财旺 → "真富贵命"

### C53 财运判读需看大运

**实战**:
- 走印比运 → 中年起步
- 走食伤运 → 才华外显
- 走财官运 → 求财高峰
- 走比劫运 → 破财风险

## 关键 takeaway

**"财运吉"标签的实际意义有限**:
- 70% 样本都是"财运吉"
- 但实际收入差异巨大
- 必须给具体信号和建议

**实战中**:
- 客户问"财运怎么样" → 看身强身弱 + 大运
- 不能直接说"财运吉"
- 必须说"财运有 X 信号,具体能否实现需 Y 条件"
