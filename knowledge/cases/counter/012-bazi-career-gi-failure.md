# 反向八字案例 010: "事业吉"判读的实际贫困

## 关键洞察

**玄照 CV 在 218 样本中事业判读 100% 为"吉"**——这意味着:
- 任何人都被告知"事业吉"
- "事业吉"基本无意义
- **实际可能贫困**(算法不会识别)

## 应"事业吉但实际贫困"的常见组合

```
1. 身弱财官旺 → 想做事但扛不住
2. 食神格身强 → 才华横溢但贫困(典型艺术家)
3. 命中无财星 → 有事业但没钱
4. 比劫夺财 → 合作分利,实际到手少
5. 印星坏财 → 学历高但穷酸
6. 命中凶格太多 → 多次失败
```

## 实战反例(理论推断)

### 反例 1: 高学历贫困
- **命盘**:印星旺 + 财星弱 + 食神泄秀
- **玄照判读**:事业吉(有学历有技能)
- **实际**:高学历低收入(典型书呆子)

### 反例 2: 创业失败
- **命盘**:身弱 + 财官旺 + 大运走克
- **玄照判读**:事业吉(有机会)
- **实际**:创业失败多次,负债

### 反例 3: 大器晚成
- **命盘**:前 30 年大运走克,40 岁后转好
- **玄照判读**:事业吉(终身事业吉)
- **实际**:前半生贫困,40 岁后翻身

## 反向准则

### C29 玄照"事业吉"基本无意义

**应对**:
- 永远不要单独看"事业吉"标签
- 必须看具体的:行业方向 + 升迁可能性 + 创业适配 + 收入水平
- 给客户讲事业时,**用"有空间"代替"吉"**

### C30 事业吉 + 财运吉 = 可能仍是普通人

**实战**:
- 事业吉 + 财运吉 + 健康凶 = "工作狂但身体不好"
- 事业吉 + 财运凶 = "有名无利"(这种情况玄照不出)
- → 必须看多维度组合,不能单看一个维度

## 关键 takeaway

**玄照事业判读 = "有方向" 而非 "会成功"**:
- 实战中:事业"吉"→ "有方向/有机会"
- 事业"凶"→ 真正有问题(玄照不出"事业凶")
- 客户问事业 → 必看其他维度,不要单看事业
