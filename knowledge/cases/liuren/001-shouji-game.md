# 大六壬案例 001: 深圳李某 2013 年手机游戏合作预测

## 求测者信息

- **关系**:深圳李某(男,IT 从业者)
- **事项**:2013 年 2 月起课,预测与人合作开发手机游戏能否成功
- **起课时间**:2013 年 2 月
- **求测动机**:是否值得投入做手机游戏开发合作
- **结果**:项目运作起来但 9 月撤出,合作伙伴理念不同开始内耗,**秋天散伙**

## 作者判读(原文摘要)

**课象**:
> "交车合,最宜合作" — 表面看,课象显示合作适宜启动

**断语**:
> "项目虽然可以运作起来,但最终估计难以成功,**秋天散伙**"

**逻辑链**:
- 交车合 = 启动合作(应期近)
- 但项目本身课象带耗散 → 长期难成
- 秋天(申酉月)金旺冲克木(项目主体) → 散伙应期

## 应验验证

| 预测 | 实际 | 命中 |
|---|---|---|
| 合作可启动 | 合作启动 | ✅ |
| 最终难成 | 9 月撤出,内耗 | ✅ |
| 秋天散伙 | 9 月(秋初)撤出 | ✅ 命中 |

## "为什么会变成这样"的因果链

```
大六壬断合作成败的核心逻辑:
1. 起课 → 排三传四课
2. 求测事件 → 取用神(合作 → 财爻代表项目收益)
3. 课象看:
   - 交车合 → 启动无碍
   - 财爻空亡/死绝 → 长期难成
   - 妻财/子嗣皆空 → 散伙应期
4. 应期 = 财爻受冲克之月 → 秋天(申酉月)金旺冲木
```

## 判断准则提炼

- **R1 交车合(发动合作) + 财爻空亡** → 启动易,长期难
- **R2 秋天(申酉月)金旺** → 木类项目应灾(手机游戏属火木,克泄交加)
- **R3 合作伙伴"理念不同"** = 课象中兄弟爻(同伴)克财爻(项目) → 内部争夺致散

## 反例风险

- 单案例无法证明"交车合 = 启动易"
- 财爻空亡未必都应"秋天散伙"
