# 大六壬案例 002: 2022-11-23 男命戊辰测婚

## 求测者信息

- **起课时间**:2022 年 11 月 23 日
- **男命**:戊辰
- **女命**:乙亥
- **求测事项**:婚姻(女命是否有外遇、是否离婚、应期)
- **结果**:女命老公有外遇,预测离婚时间"应在明年正月",实际反馈"等不到明年正月了马上就要去离"——**部分命中,应期提前**

## 作者判读(原文摘要)

**断语**:
- 女方为家庭主妇 ✓
- 老公有外遇 ✓
- 性格特征命中 ✓
- 离婚应期"明年正月" ✗(提前了,实际 2022-12 / 2023-1 之间)

## 应验验证表

| 预测 | 实际 | 命中 |
|---|---|---|
| 女为家庭主妇 | 是 | ✅ |
| 老公有外遇 | 是 | ✅ |
| 离婚 | 离 | ✅ |
| 离婚应期"明年正月" | 提前到 2022 年底/2023 年初 | ⏳ 应期偏差 |

## "为什么会变成这样"的因果链

```
大六壬断婚姻的核心逻辑:
1. 取用神(婚姻 → 妻财爻代表妻子/婚姻状况)
2. 妻财爻空亡/被冲 → 婚姻动荡
3. 官鬼(老公)临桃花/驿马 → 老公有外遇
4. 应期 = 妻财爻被冲克之月
5. 此例"明年正月"为推断,但实际应期提前
```

## 判断准则提炼

- **R4 妻财空亡 + 官鬼临桃花** → 婚姻危机 + 配偶外遇信号
- **R5 应期判断的局限性** → 课象可能给出"应期范围"(正月前后),但实际应期可能提前

## 反例风险 / 局限性

- **应期精度**比事件方向更难
- 同一课象,不同起课时间可能得出不同应期
- 求测者心态和事态发展速度会影响应期

## 关键 takeaway

**事件方向比应期精度更可靠** — 这个案例告诉我们:
1. 大六壬断"是否离"、"配偶是否有外遇"很准
2. 但断"何时离"会偏差(1-3 个月甚至更多)
3. 实战中:**给客户讲方向,给"应期窗口"而非精确日期**
