# 神峰通考 卷五

## 一、原文摘要

卷五由四个大板块构成:

1. **论大运/论太岁** — 张楠短篇,论大运起法、月令用神、太岁主君之尊。六十甲子运行之轨则。
2. **认格局生死之歌** — 综述各格的"必死"条件。印绶、正官、伤官、拱禄拱贵、日禄归时等的死地。
3. **五星论** — 五行与五方五帝的对应,详论金木水火土在四时的旺相休囚。
4. **金不换看命绳尺** + **金不换骨髓歌断** — 六十日干(甲乙丙丁戊己庚辛壬癸各六生月)逐月喜忌方向,是后世民谣派命学的总源头。

立意:本卷从格法转向**五行气化与四时旺相**,为下一卷的歌赋体系铺路。

## 二、关键章节逐字引文

### 引 1: 论大运

> 夫大运者,就月上起。譬之树苗,树之见苗则名。月之用神,则知其格,故谓交运。如同接木然,论命有根苗化实者,正合此意,岂不宜哉。或甲乙得寅卯运,名曰动劫财败财,主克父乃克妻破财争斗之事。行丙丁巳午运,主吉庆增产。辰戌丑未戊己财运,主名皆通。

(原位: 论大运,pos 988)

## 三、字句疏解

**"大运就月上起"** ——大运的起算点不是以日为主,而是以月令为主。月令是命局的"用神来源",所以大运起于此,用以观察用神的旺衰流转。

**"月之用神,则知其格"** ——这是卷五第一句方法论:看月令用神就知是何格,起运即是观察用神盛衰的开始。

**"太岁乃天中天子,甚不可犯之"** ——太岁在天干地支中代表"君位",与太岁相冲、相克谓之"犯岁君"。"五行有救,其年反必为财"——若八字有制,反主该年发财(如甲日见戊太岁,若有庚制戊,反主财来)。

**"印绶见财行财运,又见死绝入黄泉"** ——印绶格最怕行财运,又入死绝之地,大凶。这一句成为后世"贪财坏印"论的源头。

**"正官见杀及伤官刑冲破害"** ——正官格最怕七杀混杂(官杀混)、伤官克官、刑冲破害。三者见一已凶,岁运并临则必死。

**"金星论"的核心** ——
"春月见金性柔体弱"——春天金被木盗气,不可专用;
"夏月之金性尚在柔"——夏火克金,金未得位;
"秋金当权乘胜"——秋金当令最强,但"金多愈刚,刚而必折"——金太旺反而折,需火锻;
"冬金形寒性冷"——冬金死绝,但火多能"性温体健",能成器。

**金不换看命绳尺** ——这是张楠所收的"民谣派命学"精华,四言四句一组,涵盖十二种典型命运格局。"此术莫轻传,泄漏天之机"——张楠也认同此赋"近天机"。

**金不换骨髓歌·六十日干喜忌** ——
六十日干(十天干各六生月)逐月给出顺逆行运的吉凶。**核心规则**:
- 身旺时喜行财官;
- 身弱时喜行印比;
- 太过时喜行泄;
- 不及时喜行扶;
- 看月令透出何字定用神;
- 顺逆看月令阴阳(阳年男顺女逆,阴年男逆女顺)。

**"顺行不似逆行高"** ——这是骨髓歌的"基调":很多时候,逆行运反而比顺行运更显贵。**条件**:原局用神为"逆向"用神时(如月令财星,日主身强需取西北金水运),逆运为上。

## 四、实战规则 / 玄照引擎调用规范

### 4.1 大运起算接口

```
大运起算:
  阳年男命 -> 顺行(月令后第一个节气为起点)
  阴年男命 -> 逆行
  阳年女命 -> 逆行
  阴年女命 -> 顺行
  起运岁数:从生日到节气的天数除以3,即得起运岁数

大运运行总纲:
  IF 月令为用神来源 THEN:
    IF 大运为月令同五行 -> 旺,吉
    IF 大运克月令 -> 凶(破格)
    IF 月令生大运 -> 吉(进)
    IF 大运克月令 -> 凶(退)
  详见各格接口
```

### 4.2 太岁接口

```
太岁交互:
  IF 大运冲太岁 -> 凶(运冲岁甚不利)
  IF 太岁冲大运 -> 吉(岁冲运吉)
  IF 日干冲太岁 -> 凶(下犯上)
  IF 太岁冲日干 -> 凶轻(上凌下)
  IF 太岁生日干 -> 吉(君恩臣)
  IF 日干克太岁 -> 看是否有救:
    IF 有救 -> 反主财
    IF 无救 -> 死

甲乙日主冲犯太岁的特例:
  甲乙日见庚辛太岁 -> 天干相冲
  IF 生寅卯亥未日时 -> "决死无疑"
  IF 有印化之 -> 反吉
```

### 4.5 金不换骨髓歌·调用逻辑

```
金不换骨髓歌调用:
  1. 取日干 + 生月(以节气为准,非阴历月)
  2. 查"X X X月"对应四言句,定顺逆行运
  3. 关键判断项:
     - 财官是否透干 -> 决定行运喜财还是喜官

  4. 输出"顺行/逆行"的吉凶提示
```
