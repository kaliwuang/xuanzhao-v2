# 神峰通考 卷二

## 一、原文摘要

卷二由四个主格与一个十论构成:

1. **时上一位贵格**——专论时柱七杀一位透出的格局。核心:身旺能任则贵,身弱则贫。
2. **附官杀去留杂格**——补充《喜忌篇》"去官留杀""去杀留官"的具体方法。
3. **月支正财格 / 弃命从财格**——以月支藏财为本,日主旺则任财,日主弱则弃命。
4. **时上偏财格 / 月偏财格**——以时柱偏财透出为用,身旺多富贵,身弱则破财伤身。
5. **伤官食神格 + 伤官十论**——伤官分真伤、假伤,日干旺弱决定喜忌方向;十日各有喜忌歌诀。

立意:本卷进入"格法实战",每格先述理论,再附张楠亲验命例。

## 二、关键章节逐字引文

### 引 1: 时上一位贵格开篇

> 楠曰:时上一位格,盖取时一点杀星。若日干生旺,时上有杀,则用之为时上一位贵。若身旺杀衰,喜杀旺运,富贵多子,盖杀乃子星也,身旺能任其子也,若日干弱,时上杀旺,怕行杀旺及财运,正谓时逢七杀赶身衰,则主贫贱无子。杀能克身,不能生子,正谓时逢七杀本无儿。

(原位: pos 266–267)

### 引 2: 时上一位贵·补注(化杀与制杀)

> 食神制杀,化鬼为官,因宜权贵。所谓食居先,杀居后,功名两全是己。羊刃合杀,变凶为吉,亦能权贵,所谓甲以乙妹妻庚,凶为吉兆中是已。
> 食神固能制杀,而伤官亦能制杀,便伤官不如食神之力。夫羊刃固能合杀,而伤官亦能合杀,但伤官不如羊刃之势显。

(原位: pos 269–271)

### 引 3: 时上一位贵·阴日伤官制杀

> 阳日伤官能制杀,而不能合杀,如甲日见丁为伤官,能制庚金之杀,而不能合庚是己。阴日伤官能合杀,而自克制杀,如乙日见丙为伤官,难合辛金之杀,自能制辛是已。

(原位: pos 271,阴阳日主伤官区别)

### 引 4: 时上一位贵·古歌

> 时上偏官一位强,本身健旺富非常。年月并无官财杀,独于时位最相当。
> 又曰:时土一位贵,藏在支中是。日主要刚强,利名方有气。
> 又曰:时逢七杀本无儿,此理人当仔细推。岁月时中知有制,定知有子贵而奇。

(原位: pos 273–285)

### 引 5: 月支正财格·开篇

> 正财者,何也?财为吾性命之物,人见之未尝不欲。若身主有气,则能任之,若金宝,若田产,皆我之物也。身弱则不能任,如盗偷人财物,事发则为害命之物也。书云:"逢财喜杀而遇杀,十有九贵。"

(原位: pos 312)

### 引 6: 弃命从财格

> 弃命从财格,此则不论阴阳日主皆从也。财乃吾妻,身主无力,不能任其财也,只得舍命而从之。如人自己无主,只得入于妻家,就要生起财星,而亦畏身入财,乃印生之地,即同弃命从杀而论,理出于正。

(原位: pos 313)

### 引 7: 时上偏财格

> 时上偏财格,盖以日干有气,能任其财也。如甲寅日见戊辰时也,天干透出财神,斯格方真。若又岁月有财相杂,克倒曰干,则不能任其财,嫌官杀运,克倒曰干,则不能任其财。盖身太旺比肩多,又喜官杀运制去其比肩,放起其财神。

(原位: pos 343)

### 引 8: 偏财忌比劫·古歌

> 偏财本是众人财,最忌支中比动来。身强财旺皆为福,若带官星更妙哉。
> 又《古歌》云:时上偏财一位佳,不逢冲破享荣华。败财劫刃还无遇,富贵双全比石家。

(原位: pos 344–346)

### 引 9: 张楠自造(希文自造)

> 己酉 丙子 壬寅 乙巳 希文自造时月偏财类聚之格
> 楠曰:壬水生逢子月天,偏财类聚喜周全。壬申再转南方运,火土重逢福禄坚。
> 予修净土人也,亦不事此术,为继家君之志,而编此书。予命正合书云:"素食慈心,印绶遂逢于德。"信矣……然命至微妙,宣父难言之。

(原位: pos 349–351)

### 引 10: 伤官食神格·开篇

> 伤官食神,一阴一阳谓。伤官阴见阴,阳见阳之谓。食神皆盗我血气之物也。子平书论伤食神之理,书虽甚多,但所言皆不亲切。何以为之伤官也?盖人之身,以官星为管我之官,如府县官之类也……今则伤官者,则是伤杀其官,不服官管,如弑杀上官之类,则为强贼化外之民。

(原位: pos 363)

### 引 11: 伤官真假之分

> 然伤官之格,有真伤官,有假伤官。如真伤官者,甲乙日生于巳午未月,真火为伤官用事,盖甲乙日被火焚其精英,若火多而木性失,则喜北方水运,以破其伤官,扶起其木气。
> 又曰:假伤官行伤官运发,若行南方火运。动其虚火,且木气坚朴,又得火以泄其精英,金主富贵。若行北方运,破其盛火,正谓假伤官印运必死,真伤官印运必死,真伤官行伤官运必灭。

(原位: pos 365)

### 引 12: 伤官十论(十日伤官喜忌歌)

> 甲木伤官寅午全,火明木秀利名坚。运行最怕财官旺,见戌行来阻寿元。
> 乙木伤官火最强,运逢官杀转为良。只怕水多伤不尽,终身名利有乘张。
> 丙火伤官燥土重,运行财旺福兴隆。如逢水运遭伤灭,世态纷纷总是空。
> 丁火伤官火又柔,生人骄傲有机谋。运逢印绶连官杀,唾手成家孰与俦。
> 戊日伤官最怕金,柱中格畏木来侵。金衰不喜行财运,土既消磨金又沉。
> 己日伤官金最旺,弱金柔土喜财乡。运逢官杀终身福,名利兴衷不久长。
> 庚日伤官喜见官,运逢官杀贵无端。正是顽金逢火炼,少年折桂上金蛮。
> 辛日伤官申子辰,伤官伤尽喜财星。东南顺运滔滔好,见官见杀返为仇。
> 壬水伤官喜见官,合起羊刃武贵攀。若遇酉戌冲出格,声名震地不全完。
> 癸水伤官怕见官,最嫌戊己透天干。再行财旺生官地,世事劳形祸百端。

(原位: pos 375–385,引壬水论据卷中"补曰"段补足)

## 三、字句疏解

**"时上一位贵"**——专指时柱天干一位七杀透出,且月日年柱无官杀混杂。四柱官星不可再见,日主要自旺(如甲寅日)。身强杀浅则贵,身弱杀重则贫寒。

**"去官留杀"**——天干透出者易去,月支所藏者难去。去官杀之物是伤官、食神。看官杀孰轻孰重,众而有力方能去。

**"贪合忘杀"**——如六癸日干头见己字(七杀),再见甲字(己家合神),则合去己字,七杀反为我用。这是阴日以伤官合杀的典型。

**"阳日伤官制杀不能合,阴日伤官能合能制"**——甲日见丁只能制庚不能合庚;乙日见丙既难合辛金又能制辛金。这是阴阳日主在伤官制杀上的根本区别,后世论命经常混淆。

**"弃命从财"**——日主无一点生气,满局财星,则"舍命从之"。这与弃命从杀同理。但仍畏身入财的印生之地,因为印能扶身破财。

**"真伤官"与"假伤官"**——甲乙木生于巳午未月为真伤(火焚木精英);生于正月寅卯月见火为假伤(虚火未炽,木性仍坚)。真伤喜水,假伤喜火。

**"木火见官官要旺,金水伤官喜见官"**——木火假伤时木气坚朴,可任官星;金水伤官时水气泄金,喜火官暖金。但水多金弱则亦畏官,要具体看旺弱。

## 四、实战规则 / 玄照引擎调用规范

### 4.1 时上一位贵格判断流程

```
判断顺序:
1. 看时干是否一位七杀透出
2. 看月日年是否再有官杀(混入则格破)
3. 看日主是否自旺(如甲寅、甲子之类)
4. 看是否有制伏(食神制、羊刃合、伤官合)
5. 制伏多少(1位杀配1位制为佳;2-3位制则太过,为贫儒)
6. 大运:原无制走制伏运;制太过走杀旺运
```

### 4.2 正财格 vs 偏财格核心差异

| 维度 | 正财 | 偏财 |
|------|------|------|
| 阴阳 | 阴见阳、阳见阴 | 阳见阳、阴见阴 |
| 性质 | 我妻、稳定 | 众人财、横财 |
| 喜 | 身旺、比劫 | 身旺、官星 |
| 忌 | 官星(盗气) | 比劫(分夺) |
| 富贵 | 多不及 | 多富贵 |

### 4.3 伤官格引擎接口

```
伤官格调用:
  IF 时柱伤官:
    IF 日主旺 AND 伤官轻 -> flag="假伤官", action="行伤官运发"
    IF 日主旺 AND 伤官重 -> flag="真伤官", action="行北方水运/印运破伤"
    IF 日主弱 -> flag="伤官泄气太过", action="急须行印运/财运救身"
  IF 伤官见官:
    IF 有财有印 -> "可解"
    ELSE -> "祸百端"
  IF 食神格:
    IF 一位食神 -> "食神生财格,喜财星"
    IF 食神多 -> "等同于伤官,不畏印运"
    IF 食神被枭印夺 -> "寿夭贫寒"
```
