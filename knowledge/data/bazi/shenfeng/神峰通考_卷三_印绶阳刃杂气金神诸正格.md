# 神峰通考 卷三

## 一、原文摘要

卷三由四个大格构成:

1. **印绶格(含偏印枭神)** — 月生日干者为印,母旺子弱赖印扶身;印多见财则破。子格:阳刃、专财、日德、日贵、魁罡、六壬趋艮、六甲趋乾等。
2. **阳刃格附比劫、建禄格** — 六阳日干临禄前一位为刃。六阴无刃,乙丁己辛癸无刃。刃为劫财之器,要官杀制之则吉。
3. **杂气财官印绶格(附时墓格)** — 辰戌丑未四库藏蓄,喜冲喜露,忌干支压伏。本卷实质上把卷三剩余格全部塞进"杂气"。
4. **金神格** — 甲己日金神三种时(癸酉、乙丑、己巳),柱有火局制伏方贵,水运最凶。

立意:本卷主论"子平正格"中的印绶、阳刃、杂气、金神四大格,为后续外格(卷四)做铺陈。

## 二、关键章节逐字引文

### 引 1: 印绶格·开篇

> 正印偏印格者,如父母生身之义也,盖日主得其资助。书云:"印绶生月利官运,畏入才乡,盖财乃破印之气也。"此亦书之死路,非通变之道也。然四柱印星太旺,日主有气,印叠生身,如人元气本旺,再服补药,生可存乎?此则几用见屠以破印也。四财少,必须运上财神为吉。又若日主根轻,印星又弱,最畏财星,谓之贪财损印,宜也。

(原位: pos 482)

### 引 2: 真印假印论

> 又有真印假印,如丙日生人,生临亥月,或用亥月甲木作假印地也。十月木气,根桔叶落,则此衰木,宜行东方木旺之地,以辅助其根气,则枯苗得雨,勃然而兴,畏巳酉丑运,冲克其木,尤畏行西方申辛本,天干地支俱全,损伤尤甚。若天干得壬癸甲乙丙丁盖头,虽祸亦浅,又若丙丁日主,临寅卯多根之地,谓之真印也。

(原位: 印绶格中部)

### 引 3: 印绶·诸格歌诀

> 月逢印绶喜官星,运入官乡福必清。死绝运临身不利,后行财运百无成。
> 印绶官星旺气纯,伤官多遇转精神。如行死绝并财地,无救反为泉下人。
> 印星偏者是枭神,柱内最喜见财星。身旺遇之方是福,身衰枭旺更无情。
> 《玉匣赋》云:华盖与文星共会,尉迟为五伯良臣。

(原位: 古歌诸段)

### 引 4: 印绶·辩虚实之关键句

> 盖印不能胜金,谓之印绶被伤。倘若荣华不久,真假印辩,不可不究。财、官、印、杀、食神、伤官,此六乃日干月令。正格外有阳刃格!此系日月相通。出此之外,或虚邀财官,或刑合财官,或暗拱财官,或冲摇财官,亦几近理。说见小文。

(原位: 印绶格末段,本卷承上启下关键句)

### 引 5: 阳刃格·甲日卯刃之病

> 阳刃格,六阳日干谓之阳刃理也,六阴日干不谓之刃。但用刃之说,未究其理,则冥然不知也。书云阳刃无冲要可极品。盖甲日生临卯月,甲见卯中乙木,如兄见弟,则能分我之祖财,夺我之祖业,再加岁日时中木又有气,何用乙木之再助我乎?如是则不用刃也,而以刃为病也,则用庚金七杀。

(原位: 阳刃格整段开篇,pos 537)

### 引 6: 阳刃·喜合忌冲诀

> 帮身羊刃,喜合嫌冲。本注曰:刃乃才身这物,大怕身旺逢之,得一重杀与刃相匐,化为权星。若见官与刃冲战,乃成恶性杀。用者当审其轻重,好恶何如。

(原位: 阳刃格中部)

### 引 7: 建禄格·核心句

> 建禄格者,日主得主禄之地,非官禄也。书云月令建禄,多无祖屋,而不显言其理,人用财为马,官为禄,假如甲日生寅月,甲以辛为官,己土为财,财官到寅为死绝地。

(原位: 阳刃附建禄格,pos 539)

### 引 8: 建禄·喜见财官例(古韩侍郎造)

> 庚子 戊子 癸丑 庚申 古韩侍郎造
> 壬申 丙午 己亥 庚午 赵丞相之造
> 古见杀有制例 辛丑 庚寅 甲辰 乙亥 贺丞相之造
> 木旺春令,日干头官杀相连,只论杀,运行丙戌丁,制杀乡,宜为大贵也。

(原位: 建禄格末段)

### 引 9: 杂气财官印绶格·本卷最重一段

> 杂气财官者,盖辰戌丑未四字,乃天地不正之气,为天地四个牢狱之所,又为天地四个收藏之库。如丙丁日生辰月,若天干透也戊己多,则作杂气火土伤官格;如透出乙木多,则作杂气印绶格;如透出癸水多,方作杂气财官格。

(原位: 杂气财官印绶格开篇,pos 604)

### 引 10: 杂气·喜见冲与不可一概而论

> 又曰:杂气财官喜见冲。惟日干旺,用杂气财官者,喜见之,原财藏天地之库中,牢密坚固。如戊日生辰月,癸乙为财官,钱在库中,十分牢固,若无戌时锁匙,岂能开之,冲开财库,福兴隆然也。若不用财官,则不可把冲开作论。当以中和偏枯看,或嫌其冲为害神,再冲也来,其害愈甚。

(原位: 杂气财官格中段)

### 引 11: 杂气·伤官不同论

> 又如丙丁日生戌月,透出戊土多,作杂气伤官格看,原日干丙丁属火,到九月授衣之月,火气寒凉,见土多泄弱火之精,喜东主木运,克去其伤官,不来泄我,喜木运又来资我则吉。若到辰运,再冲出戌中戊土来,愈泄弱我精神,安得不死,此亦不可作杂气财官喜见冲也。

(原位: 杂气格末段)

### 引 12: 杂气古歌·"库门开透露"

> 杂气财官印月官,天干透露始为丰。财多官旺宜冲破,切忌干支压伏重。
> 又曰:辰戌丑未为四季,印绶财官居杂气。干头透出格为真,只论财多为尊贵。
> 又曰:月令提纲不可冲,十冲九命返为凶。惟有财官逢墓库,运行到此返成功。

(原位: 古歌三段,pos 606-614)

### 引 13: 金神格·甲己二日之分

> 楠曰:甲日金神格,取癸酉乙丑己巳三个时。书曰:"甲日金神,偏宜火制。"又曰:"金神遇火贵无疑,金火灾殃有之,此格多生大贵。"但论之未明显。但甲日四柱气旺,金神又或有一二点者,遇火制之极贵。此即同时上偏官格一样看。若金神气轻,柱中有火制之太过,大畏火乡太制其金神,又喜金水以助其金神出。

(原位: 金神格开篇,pos 647)

### 引 14: 金神格·乙己二日之时

> 乙日金神,乙丑、己巳、癸酉三时也,主多贵。己日见巳酉丑金,己日金而泄其气也,即时上假伤官也。何以言之?书云:"己日金神,偏嫌火制。"若己日干精英涵蓄太过,望见此三时,金清而且秀,己土贪生而泄其秀气也。若原滞火金神气衰,最畏火而伤之。喜金运以助金神,喜水运破火,存起金神,多主富贵。

(原位: 金神格乙己段,pos 648)

### 引 15: 金神古歌

> 金神巳酉丑之时,杀刃中来真贵人。运气最宜逢火局,水乡相见祸临身。
> 又曰:性多狠暴才明敏,遇木相生立困穷。制伏运行运火局,招迁贵显富显穷。

(原位: 古歌两段,pos 650-651)

### 引 16: 金神·高谷南少卿命造

> 丁亥 癸丑 己未 癸酉 金溪高谷南少卿己日金神土金假伤官
> 楠曰:日干己未坐金神,土厚金轻理自真。运转西方金旺地,金轻岂作等闲人。
> 己未日主,生于丑月,两土气厚,凡身旺多好泄也。见酉丑结成金局,用金为假伤官。又为己日金神,盖喜有甲乙木之官杀,为伤官格之病,所以运行西方金运破木,其贵宜矣。

(原位: 高谷南命例,pos 652)

## 三、字句疏解

**"印绶生月利官运,畏入才乡"** ——"才"通"财"。张楠指出:印格之人,最利行官运(官生印、印扶身,有连环相生之妙)。最忌入财运(财破印)。但他紧接说"此亦书之死路,非通变之道"——这是张楠方法论的灵魂:经典法则不能执一,要看日主与印星的相对强弱。

**"真印"与"假印"** ——丙丁日临寅卯,根深是真印,喜行食神伤官泄秀;丙日临亥,以亥中甲木为假印,十月根枯,要行东方木旺之地补根。区别在于:真印身强可担官杀,假印身弱要行比劫印地。

**"枭神"** ——偏印的另一名。甲生亥月、乙生子月之类为偏印,无食则偏印为枭。枭神遇财则吉("偏印遇财乃发"),身旺遇之方是福;身衰枭旺则无情。

**阳刃的本质** ——甲日见卯,甲乙为兄弟,乙木分甲木之财,故为劫财之器。阳刃格最忌身旺无制,身旺逢刃必须有杀(庚金)制之,化为权星。甲刃配庚杀,刃杀双显即大贵。

**"天元一气"反衬"喜合嫌冲"** ——补注云刃与杀合化为权星,刃与官冲则恶杀,这是阴阳日主在制伏上的根本不同。"畏财"则羊刃格最忌财,刃能夺财、财能坏印,双重受制。

**建禄与阳刃的区别** ——同是月令得禄,建禄身自旺,阳刃身太旺;建禄要"用财官作身旺之托",阳刃要"杀制刃化为权星"。这与月令用神是正官还是七杀直接相关。

**杂气财官"冲开"理论** ——"如戊日生辰月,癸乙为财官,钱在库中,十分牢固,若无戌时锁匙,岂能开之。" 这是杂气格的核心比喻:**戌冲辰是开库钥匙**。但张楠强调:"若不用财官,则不可把冲开作论"——同一八字,若用印而非用财,冲则破印而非开库,需要看透用神再谈冲。

**金神三时** ——甲日金神取**癸酉、乙丑、己巳**三时;乙日金神同;己日金神同。"原即伤官格,以金为假伤官",金神其实是金假伤官的特殊命名,本质与伤官无异,喜火制金,忌水泛金。

**金神与时上一位贵的关系** ——金神格"此即同时上偏官格一样看",这是张楠将所谓"金神"还原为子平正格的尝试:**金神就是时上一位偏官,只是换了个名字**。这与卷一驳"魁罡""日贵"时的逻辑一致——凡不能还原到正格而单凭名字论命的,都不可信。

## 四、实战规则 / 玄照引擎调用规范

### 4.1 印绶格调用接口

```
印绶格:
  IF 月支生日干:
    IF 印绶成格:
      IF 日主弱+印旺 -> "印扶,喜官印连续,忌财破"
      IF 日主强+印旺 -> "印叠太过,要财星破印"
      IF 日主弱+印弱 -> "贪财损印,忌行财乡"
    IF 真印(丙丁生寅卯) -> "身强可任官杀,喜食伤泄秀"
    IF 假印(丙生亥月甲木) -> "喜东方木地补根,忌西方金地冲破"
  IF 枭神(偏印为枭):
    IF 身旺+有财 -> "偏印遇财乃发"
    IF 身衰枭旺 -> "枭神兴而早年夭折"
  IF 印+官 -> "官印相生,喜官乡"
  IF 印+财 -> "贪财坏印,大忌"
  IF 印+食 -> "印绶夺食,主寿夭"
```

### 4.2 阳刃格与建禄格核心差异

| 维度 | 阳刃 | 建禄 |
|------|------|------|
| 日干 | 阳干 | 任何日干 |
| 月支 | 禄前一位(如甲卯、丙午) | 本禄(如甲寅、丙巳) |
| 喜 | 杀制刃(刃杀双显) | 财官 |
| 忌 | 财(刃夺财)、冲 | 会杀(切忌) |
| 性格 | 刚暴、勇毅、克妻 | 自旺、得祖荫少 |

### 4.3 杂气财官格判断流程

```
杂气格判断:
  1. 看月支是否为辰戌丑未
  2. 看四库所藏之五行透出天干何字
     - 透财->杂气财格
     - 透官->杂气官格
     - 透印->杂气印格
  3. 看日主旺弱:
     - 日干旺+用财官 -> 喜冲开库(如戌冲辰为开库钥匙)
     - 日干旺+用印 -> 忌冲(冲则破印)
     - 日干弱 -> 不论冲开,只论中和
  4. 大运:
     - 用财官 -> 喜行财官之乡,忌印
     - 用印 -> 喜行印比之乡,忌财破
  5. 特别禁忌:杂气不可概以冲开为美,伤官与印格"杂气"与财官杂气走向相反
```

### 4.4 杂气格六类判断表

| 月令 | 藏干 | 透干可能成格 |
|------|------|-------------|
| 辰 | 乙戊癸 | 杂气印(乙)、杂气财官(戊癸) |
| 戌 | 辛戊丁 | 杂气伤官(辛)、杂气正财(戊)、杂气印(丁) |
| 丑 | 癸己辛 | 杂气偏财(癸)、杂气正印(己)、杂气伤官(辛) |
| 未 | 丁己乙 | 杂气正印(丁己)、杂气七杀(乙) |

### 4.5 金神格调用接口

```
金神格:
  IF 日甲乙己, 时癸酉/乙丑/己巳:
    IF 柱有火局制之 -> "极贵,多主兵刑显宦"
    IF 四柱无火制 -> "宽猛不济,见木水灾祸"
    IF 火制太过 -> "喜金水助金神出"
  本质:金神就是时上一位偏官/假伤官的别名,本质与卷二"时上一位贵"无异
  大运:
    - 喜火局 -> 制金神
    - 忌水乡 -> 祸临身
    - 月令通金局或有金气方可论
```
