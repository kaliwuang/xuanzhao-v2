# 滴天髓 · 通神论 · 天道

## 一、原文

> **欲识三元万法宗，先观帝载与神功。**

## 二、原注(相传明·刘伯温)

> 天有阴阳,故春木、夏火、秋金、冬水、季土,随时显其神功。命中天地人三元之理,悉本于此。

## 三、任氏注(清·任铁樵)

> 干为天元,支为地元,支中所藏为人元。人之禀命,万有不齐,总不越此三元之理,所谓万法宗也。
>
> 阴阳本乎太极,是谓帝载;五行播于四时,是谓神功。
>
> 乃三才之统系,万物之本原。《滴天髓》首明天道如此。

## 五、实战规则(玄照引擎调用规范)

### 规则 1:三元体系

**定义**:
- **天元** = 四柱天干(主外象、性格特征、社会表现)
- **地元** = 四柱地支(主内象、身体、家庭、根基)
- **人元** = 地支藏干(主隐藏的本质,可调侯、可通关)

**实战**:任何命盘分析必须**三层齐看**,缺一即偏。

### 规则 2:帝载与神功的优先级

**定义**:
- **帝载** = 阴阳(先天结构)
- **神功** = 五行四时(后天流转)

**实战**:
- 论命先定**日元阴阳**(甲丙戊庚壬 = 阳,乙丁己辛癸 = 阴)
- 再看**月令五行**决定旺衰
- 阴阳决定**大运顺逆**(阳男阴女顺排,阴男阳女逆排) ← **这是已修的 P0**

### 规则 3:"三元皆备" vs "偏枯"

**任氏原文判据**:
- "命贵中和,偏枯终于有损"
- "理求平正,奇异不足为凭"

**实战**:
- 凡四柱天干地支藏干**三层都能流通** = 健康格局
- 凡四柱五行**严重偏缺**(如四戊午、四癸亥) = **偏枯**,非吉格

**关键反驳**:市面谬传"四戊午是帝王命" → 任氏明确否决:
> "谬书妄言四戊午者,是圣帝之造,四癸亥者,是张桓侯之造,究其理皆后人讹传。
> 余行道以来,推过四戊午、四丁未、四癸亥、四乙酉、四辛卯、四庚辰、四甲戌者甚多,
> 皆作偏枯论,无不应验。"

**玄照立场**:**以任氏判断为准**,不收"四柱全阳=帝王"等讹传。
