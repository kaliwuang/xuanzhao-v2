# 滴天髓 · 通神论 · 第一章 · 天道

## 一、原文

> 欲识三元万法宗，先观帝载与神功。

## 二、原注(相传明·刘伯温)

> 天有阴阳，故春木、夏火、秋金、冬水、季土，随时显其神功，命中天地人三元之理，悉本于此。

## 三、任氏曰(清·任铁樵)

> 干为天元，支为地元，支中所藏为人元。人之禀命，万有不齐，总不越此三元之理，所谓万法宗也。阴阳本乎太极，是谓帝载，五行播于四时，是谓神功，乃三才之统系，万物之本原。《滴天髓》首明天道如此。

## 四、字句疏解(关键术语)

| 关键术语 | 出处 | 现代释义 |
|----------|------|----------|
| **三元** | 原文"三元万法宗"；任氏"干为天元,支为地元,支中所藏为人元" | 天元(天干)+地元(地支)+人元(地支藏干)三层体系 |
| **万法宗** | 原文"三元万法宗" | 命学的根本方法来源 |
| **帝载** | 原文"帝载与神功"；任氏"阴阳本乎太极,是谓帝载" | 阴阳(本乎太极的先天结构) |
| **神功** | 原文"帝载与神功"；任氏"五行播于四时,是谓神功" | 五行在四时流转中的运行 |
| **三才之统系** | 任氏"乃三才之统系,万物之本原" | 天地人三才的统一体系 |
| **太极** | 任氏"阴阳本乎太极" | 阴阳未分之前的本源状态 |
| **春木、夏火、秋金、冬水、季土** | 原注 | 五行在四时(加季月)的当令分布 |

## 五、实战规则 / 玄照引擎调用规范

> 本章为《滴天髓》开篇总纲,任氏注篇幅较短,仅明示"三元"+"帝载神功"两层结构。具体操作细则散见后续章节(第二章地道、第三章人道、第四章知命等)。本章可落实的实战规则如下:

### 规则 1:三元体系(本章任氏原文判据)

**任氏原文判据**:
- "干为天元,支为地元,支中所藏为人元"
- "总不越此三元之理"

**实战**:
- **天元** = 四柱天干(年、月、日、时四天干)
- **地元** = 四柱地支(年、月、日、时四地支)
- **人元** = 地支所藏之干(每支藏 1-3 个干,依本气、中气、余气取用)
- 任何命盘分析必须**三元齐看**,缺一即偏

### 规则 2:帝载 = 阴阳,神功 = 五行四时

**任氏原文判据**:
- "阴阳本乎太极,是谓帝载"
- "五行播于四时,是谓神功"

**实战**:
- **帝载层**:先定日干阴阳(甲丙戊庚壬 = 阳;乙丁己辛癸 = 阴)
- **神功层**:再判月令五行旺衰
- 二者关系:帝载(阴阳)是先天结构,神功(五行四时)是后天流转——**先阴阳后五行**

### 规则 3:三才统一

**任氏原文判据**:
- "乃三才之统系,万物之本原"
- "《滴天髓》首明天道如此"

**实战**:
- 本章定位为**总纲章**,只交代方法论起点,不展开具体格局判断
- 具体格局、八格、神煞等细则,**须引用后续章节**(如十八、中和;十二、八格 等)作补充
