## 三、流派差异(重要)

> **子平真诠派(格局派)** vs **神煞派** 之争:
> - 沈孝瞻(《子平真诠》):**重格局,轻神煞**。神煞为"雕虫小技"。
> - 万民英(《三命通会》):**神煞并重**。认为神煞可断**细节**(职业、灾祸、婚期)。
> - 现代应用:**格局为主,神煞辅佐细节**(如同柱天乙+正官 = 贵人提拔;羊刃逢冲 = 血光)。

**玄照立场**: 神煞作为**辅助验证手段**,用于:
1. 验证格局推断
2. 推断**事项细节**(年份、方位、关系)
3. **不可单独用**——无格局之神煞多为"虚神"

## 四、古籍争议条目(慎用)

| 神煞 | 争议 | 玄照立场 |
|------|------|---------|
| 桃花 | 《滴天髓》认为"桃花为忌",《三命通会》认为"桃花主贵" | **因位置而异**:时柱桃花主外遇,年柱主祖辈,见吉化则主艺术天赋 |
| 驿马 | 流派无争议,但**逆行/顺行**有分歧 | **统一用顺行**(主流) |
| 天乙 | 与"太极贵人"易混 | 见太极条目 |
| 魁罡 | 《渊海子平》:"魁罡日最凶,不可冲" | **过激**,现代应用逢冲未必凶 |

## 五、玄照神煞引擎调用规范

```python
# 引擎层统一调用接口(伪代码)
from knowledge.data.bazi.shensha_index import lookup

def get_shensha(chart) -> dict:
    """返回 {shensha_name: {position: str, level: str, source: str}}"""
    return lookup(chart.day_gan, chart.year_zhi, chart.month_zhi, chart.day_zhi, chart.hour_zhi)
```

**调用约束**:
- 每次排盘必须返回**所有 45 种的存在状态**(有/无)
- 单条神煞的**位置**(年/月/日/时柱)必须准确
- **优先级**: 格局 > 神煞 — 神煞不能颠覆格局结论
