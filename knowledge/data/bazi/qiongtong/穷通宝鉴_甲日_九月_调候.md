## 一、原文

```
书曰：甲乙生寅卯，庚辛干上逢，离南推富贵，坎地却为凶。



三夏甲木

四月甲木退气，丙火司权，先癸後丁。

庚金太多，甲反受病。若得壬水，方配得中和，此人性好清高，假装富贵。即荫袭显达，终日好作祸乱，善辨巧谈，喜作诗文。此理最验。
```
