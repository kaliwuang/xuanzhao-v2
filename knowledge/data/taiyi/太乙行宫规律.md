# 太乙行宫规律

太乙行宫是太乙神数的核心推算方法，指太乙在九宫中按特定规律运行，决定不同年份、时期的运势基调。

## 一、太乙行宫基本规律

### 1. 行宫周期
太乙行宫以**360年**为一个大周期，每**40年**行一宫，按九宫顺序循环运行。

### 2. 行宫顺序
太乙行宫顺序遵循洛书九宫轨迹：
**一宫乾 → 二宫离 → 三宫艮 → 四宫震 → 五宫中 → 六宫兑 → 七宫坤 → 八宫坎 → 九宫巽 → 回到一宫乾**

### 3. 起始位置
太乙行宫起始于一宫乾（西北），从特定基准年份开始推算。

## 二、太乙行宫推算方法

### 1. 年份推算公式
```
太乙所在宫位 = (年份 - 基准年份) ÷ 40 取余数 + 1
```
其中基准年份通常为黄帝纪元元年或特定历法起点。

### 2. 具体推算步骤
1. 确定所求年份
2. 计算与基准年份的差值
3. 除以40取余数
4. 余数加1即为太乙所在宫位数
5. 若结果大于9，则减去9

### 3. 举例说明
假设基准年份为公元前2697年（黄帝元年）：
- 求2024年太乙所在宫位
- 差值：2024 - (-2697) = 4721
- 4721 ÷ 40 = 118余1
- 余数1 + 1 = 2
- 因此2024年太乙在二宫离

## 三、太乙行宫的运势影响

### 1. 吉宫行运（一、三、五、七、九宫）
- **一宫乾**：权威时期，利开创、领导
- **三宫艮**：稳定时期，利守成、积累
- **五宫中**：太平时期，政权稳固
- **七宫坤**：繁荣时期，农业兴盛
- **九宫巽**：进取时期，外交顺利

### 2. 凶宫行运（二、四、六、八宫）
- **二宫离**：是非时期，文书口舌
- **四宫震**：动荡时期，惊变频发
- **六宫兑**：破坏时期，口舌诉讼
- **八宫坎**：危险时期，险情频发

## 四、太乙行宫特殊现象

### 1. 太乙入中宫
太乙入中宫为最吉之象，主天下太平，政权稳固。但中宫为至尊之位，太乙入中宫时需特别谨慎。

### 2. 太乙对冲
太乙所在宫位与对冲宫位（如一乾对七坤，二离对八坎等）形成对冲关系，主冲突、变动。

### 3. 太乙三合
太乙所在宫位与三合宫位（如一乾、五中、九巽为三合）形成合力，主合作、助力。

## 五、太乙行宫与历史周期

### 1. 大周期划分
- 每360年为一个完整循环
- 每40年为一个宫位运程
- 每10年为一个小运

### 2. 历史验证
通过历史事件可以验证太乙行宫规律的准确性：
- 太乙在吉宫时期，多对应盛世、稳定期
- 太乙在凶宫时期，多对应乱世、动荡期

## 六、太乙行宫应用要点

1. **长期规划**：太乙行宫适合长期运势判断，以40年为单位
2. **结合八字**：太乙行宫主大环境，需结合个人八字判断个人运势
3. **动态调整**：太乙行宫每40年变化一次，需提前规划
4. **五行生克**：宫位五行与个人八字五行的生克关系影响吉凶程度
