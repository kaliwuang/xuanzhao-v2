## 一、四化的概念

紫微斗数中,**四化** = 化禄、化权、化科、化忌,共 4 种能量流动。

| 四化 | 含义 | 五行类比 |
|------|------|----------|
| **化禄** | 财禄、情感、缘起 | 水(润下) |
| **化权** | 权力、掌控、决断 | 木(曲直) |
| **化科** | 名声、文雅、贵人 | 火(炎上) |
| **化忌** | 执念、亏损、破败 | 土(稼穑) |

**关键**: 四化是**动态能量**, 不同于星曜的**静态性质**(庙陷)。

## 三、四化的实战规则

### 规则 1: 自化(同宫干飞四化)

> 当**生年四化**飞到的宫位**宫干**与四化**同五行**,称为**自化**。

**例**: 甲年生人, 廉贞化禄飞入**巳**宫(巳宫干为丙)。
- 廉贞 = 火, 丙 = 火, **同五行**
- 称为 "廉贞**自化禄**"——力量**加倍**,但**吉处藏凶**(易散)

| 自化类型 | 含义 |
|---------|------|
| 自化禄 | 吉处藏凶,易得易失 |
| 自化权 | 权不稳,易被夺 |
| 自化科 | 名不耐久,虚名 |
| 自化忌 | 凶处藏凶,**忌入忌宫大凶** |

### 规则 2: 飞化(向对宫飞四化)

> 紫微斗数中,**宫位有"本宫"与"对宫"**。四化从本宫飞向对宫,称为**飞化**。

| 飞化类型 | 含义 |
|---------|------|
| 本宫化忌飞对宫 | 我之执念影响对宫(关系紧张) |
| 对宫化忌飞本宫 | **冲我**(突发事件) |
| 三方四正会忌 | 多方夹击,压力大 |

### 规则 3: 四化叠加庙陷

> 四化**不能改变庙陷等级**,但**改变吉凶权重**。

| 组合 | 力量 |
|------|------|
| 化禄 + 庙 | **大吉**(吉上加吉) |
| 化禄 + 陷 | 微吉(吉也无力) |
| 化忌 + 庙 | **大凶**(吉处藏凶) |
| 化忌 + 陷 | **大凶中之大凶** |

### 规则 4: 大限与流年的四化

> **生年四化**管**先天命格**(一生)。
> **生月四化**管**大限**(十年一宫)。
> **时辰四化**管**流年**(一年一宫)。

**实战**: 看流年运势,**流年四化优先于生年四化**(时效性强)。

## 四、流派差异

| 流派 | 四化用法 |
|------|---------|
| **紫微派(三合派)** | 仅看**生年四化** |
| **飞星派** | 生年 + 生月 + 时辰**三层叠加** |
| **河洛派** | 用**卦气**推四化(罕见) |
| **玄照立场** | **飞星派**为主,生年定格局,生月/时辰**辅佐判断细节** |
