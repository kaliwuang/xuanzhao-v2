# 大六壬常用课格详解

## 概述

课格是大六壬起课的规则和格局，决定三传的取法和课式的吉凶。常用课格有十种，各有不同的起课条件和应用方法。

---

## 一、元首课

### 课格含义
元首课又称"一下贼上"，是四课中只有一个下贼上（地克天）的格局。代表事情单纯、明确，有主动进取之象。

### 起课条件
- 四课中只有一个下贼上
- 其余三课无克或上克下

### 取传方法
- **初传**：取下贼上之神
- **中传**：初传所临地盘之天盘
- **末传**：中传所临地盘之天盘

### 判断方法

**看初传**：
- 初传旺相：事可成，开端好
- 初传休囚：事难成，开端差
- 初传空亡：事不成或有变

**看天将**：
- 初传吉将：事有好的开始
- 初传凶将：事有不利开端

**看生克**：
- 初传生日干：有助益
- 初传克日干：有阻碍

### 应用场景
- 占求事：事有主动，可进取
- 占求财：财可得，需主动
- 占仕途：有升迁之机
- 占官司：有主动之利

### 吉凶判断
- 初传吉：事成
- 初传凶：事败
- 三传皆吉：大吉
- 三传皆凶：大凶

## 二、重审课

### 课格含义
重审课又称"一上克下"，是四课中只有一个上克下（天克地）的格局。代表事情需审慎，有被动承受之象。

### 起课条件
- 四课中只有一个上克下
- 其余三课无克或下贼上

### 取传方法
- **初传**：取上克下之神（被克者）
- **中传**：初传所临地盘之天盘
- **末传**：中传所临地盘之天盘

### 判断方法

**看初传**：
- 初传旺相：虽被动，终能成
- 初传休囚：被动无力，事难成
- 初传空亡：事不成或有变

**看天将**：
- 初传吉将：被动中有利
- 初传凶将：被动有害

**看生克**：
- 初传生日干：被动中有助
- 初传克日干：被动有害

### 应用场景
- 占求事：需审慎，不可妄动
- 占求财：财难求，需等待
- 占仕途：需谨慎，防有失
- 占官司：需防守，防有损

### 吉凶判断
- 初传吉：虽被动，终可成
- 初传凶：被动有害
- 三传皆吉：终可成
- 三传皆凶：终不成

## 三、知一课

### 课格含义
知一课又称"二上克下"或"二下贼上"，是四课中有两个同类克的格局。代表事情有选择，需取舍。

### 起课条件
- 四课中有两个上克下
- 或四课中有两个下贼上

### 取传方法
- **初传**：取与日干比和之神
- **中传**：初传所临地盘之天盘
- **末传**：中传所临地盘之天盘

### 判断方法

**看初传**：
- 初传与日干比和：事有选择，需取舍
- 初传旺相：选择有利
- 初传休囚：选择不利

**看天将**：
- 初传吉将：选择有利
- 初传凶将：选择不利

**看生克**：
- 初传生日干：选择有助
- 初传克日干：选择有碍

### 应用场景
- 占求事：事有选择，需决定
- 占求财：财有两路，需选择
- 占仕途：有两条路，需取舍
- 占官司：有两方争，需判断

### 吉凶判断
- 初传吉：选择有利
- 初传凶：选择不利
- 三传皆吉：选择正确
- 三传皆凶：选择错误

## 四、涉害课

### 课格含义
涉害课是四课中有多个克，取涉害最深之神的格局。代表事情复杂，需深入分析。

### 起课条件
- 四课中有多个克
- 需计算涉害深浅

### 取传方法
- **初传**：取涉害最深之神
- **中传**：初传所临地盘之天盘
- **末传**：中传所临地盘之天盘

### 涉害计算
从被克之神起，数到克之神止，经过的宫位数即为涉害深度。涉害最深者取为初传。

### 判断方法

**看初传**：
- 初传涉害深：事复杂，需深入
- 初传涉害浅：事简单，易处理
- 初传旺相：虽复杂，终能成
- 初传休囚：复杂难解

**看天将**：
- 初传吉将：复杂中有利
- 初传凶将：复杂有害

**看生克**：
- 初传生日干：复杂中有助
- 初传克日干：复杂有害

### 应用场景
- 占求事：事复杂，需分析
- 占求财：财路复杂，需选择
- 占仕途：情况复杂，需判断
- 占官司：案情复杂，需深入

### 吉凶判断
- 初传吉：虽复杂，终可成
- 初传凶：复杂有害
- 三传皆吉：终可成
- 三传皆凶：终不成

## 五、遥克课

### 课格含义

### 遥克类型

### 判断方法

**看初传**：
- 初传旺相：事虽远，终可成
- 初传休囚：事远难成
- 初传空亡：事不成或有变

**看天将**：
- 初传吉将：远方有喜
- 初传凶将：远方有忧

**看生克**：
- 初传生日干：远方有助
- 初传克日干：远方有碍

### 应用场景
- 占出行：事在远方
- 占求财：财在远方
- 占仕途：机会在远方
- 占官司：事涉远方

### 吉凶判断
- 蒿矢课：主远事可成
- 弹射课：主远事难成
- 初传吉：远方有喜
- 初传凶：远方有忧

- **末传**：中传所临地盘之天盘

## 八、八专课

### 课格含义
八专课是干支同位，四课只有两课的格局。代表事情专一，需专注。

### 起课条件
- 日干支同位（如甲寅、乙卯等）
- 四课只有两课
- 取干支上神

### 取传方法
- **初传**：取干支上神
- **中传**：初传所临地盘之天盘
- **末传**：中传所临地盘之天盘

### 八专类型
- **干支同位**：如甲寅、乙卯、丙午、丁巳等
- **阳日**：取干上神
- **阴日**：取支上神

### 判断方法

**看初传**：

- 初传凶将：专一有害

**看生克**：
- 初传生日干：专一中有助
- 初传克日干：专一有害

### 应用场景

- 占求财：财专一，需专注
- 占仕途：需专注，不可分心

## 十一、课格综合判断

### 课格吉凶总论

| 课格 | 吉凶 | 主事 |
|------|------|------|
| 元首课 | 吉 | 主动进取 |
| 重审课 | 中 | 被动承受 |
| 知一课 | 中 | 选择取舍 |
| 涉害课 | 中 | 复杂深入 |
| 遥克课 | 中 | 遥远等待 |
| 昴星课 | 凶 | 风险谨慎 |
| 别责课 | 中 | 特殊变通 |
| 八专课 | 中 | 专一专注 |
| 伏吟课 | 凶 | 停滞等待 |
| 返吟课 | 凶 | 反复变通 |

### 课格应用原则

1. **看课格类型**：判断事情性质
2. **看初传吉凶**：判断事情成败
3. **看三传关系**：判断事情过程
4. **看天将旺衰**：判断事情力量
5. **看空亡有无**：判断事情虚实

### 课格组合判断

- **吉课吉传**：大吉
- **吉课凶传**：先吉后凶
- **凶课吉传**：先凶后吉
- **凶课凶传**：大凶

4. 结合具体占事类型分析
